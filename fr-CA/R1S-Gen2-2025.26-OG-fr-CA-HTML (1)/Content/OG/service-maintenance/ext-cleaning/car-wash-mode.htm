Mode lavage de voiture

Le mode lavage de voiture est un paramètre que vous pouvez activer à partir de l'écran central via les commandes rapides. Lorsque le mode lavage de voiture est activé, le système de climatisation passe par défaut en mode recirculation d'air pour fermer les bouches d'aération extérieures et empêcher l'eau et la mousse de pénétrer dans le véhicule. Le mode lavage de voiture effectue les opérations suivantes :

  • Remonte toutes les fenêtres

  • Ferme les poignées de porte

  • Ferme la porte du port de charge si le véhicule ne charge pas

  • Coupe le son des alertes du capteur de stationnement

  • Empêche l'activation des essuie-glaces automatiques

  • Supprime la commutation automatique entre les thèmes Jour et Nuit

  • Supprime le verrouillage et le déverrouillage de proximité

  • Lorsque le véhicule est en mode lavage de voiture, les poignées de porte extérieures s'afficheront si vous déverrouillez le véhicule avec une clé extérieure.

  • Pour ouvrir une porte déverrouillée si les poignées ne sont pas présentes, appuyez sur l'encoche de la poignée de porte.

La vitesse du véhicule doit être inférieure à pour activer le mode lavage de voiture.
