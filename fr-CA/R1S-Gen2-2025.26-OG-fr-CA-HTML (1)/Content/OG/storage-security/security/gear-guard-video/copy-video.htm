Copier une vidéo

Pour copier une vidéo du disque dur intégré vers votre ordinateur, utilisez votre périphérique de stockage externe (non inclus).

Il peut s’agir du même périphérique de stockage externe que celui utilisé pour la Road Cam.

Le périphérique de stockage externe nécessite les éléments suivants :

  • 128 Go (minimum) de stockage

    La taille de stockage maximale est de 2 To.

  • Compatibilité USB 3.x et USB 2.0

  • Disque SSD recommandé

  • Vitesse d’écriture aléatoire de 20 Mo/s (minimum)

  • Système de fichiers FAT32

    Le véhicule demande à reformater votre appareil s’il détecte un système de fichiers différent.

Pour copier une vidéo sur votre périphérique de stockage externe, suivez ces instructions :

  1. Connectez le périphérique de stockage externe à l’un des ports USB-C de la console centrale de votre véhicule.

    N’utilisez pas de concentrateur USB ou de répartiteur.

  2. Une invite peut s’afficher sur l’écran central. Choisir Formater l’appareil pour effacer toutes les données et formater le périphérique de stockage externe afin de l’utiliser par l’application Gear Guard.

    Le formatage du périphérique de stockage externe peut prendre quelques minutes.

  3. Choisir Protection de l’engrenage dans le menu de l’écran central.

  4. Sélectionnez l’événement, puis sélectionnez la vidéo à télécharger.

  5. Choisir Télécharger  pour copier la vidéo sur votre périphérique de stockage externe.

  6. Le véhicule enregistre la vidéo sur votre périphérique de stockage externe.

Pour retirer en toute sécurité le périphérique de stockage externe du véhicule, choisissez Protection de l’engrenage dans le menu de l’écran central. Choisir Caractéristiques de la protection d’engrenage , puis choisissez Éjecter.
