Voir les vidéos enregistrées

Choisir Protection de l’engrenage dans le menu de l’écran central, ou choisissez Vidéos Gear Guard  Pour afficher les vidéos enregistrées sous les types suivants :

  • Caméra de route : Déclenché lorsque vous le souhaitez Enregistrer le clip . La Road Cam enregistre également des vidéos et enregistre des clips de la Road Cam lorsque vous klaxonnez pendant plus d’une seconde ou appuyez sur le bouton SOS . Le véhicule enregistre automatiquement les clips de la Road Cam sur votre périphérique de stockage externe.

    Pour afficher les clips Road Cam dans l’onglet vidéo Road Cam, le périphérique de stockage externe doit être connecté à l’un des ports USB-C de la console centrale.

    Pour afficher les enregistrements continus, accédez à votre périphérique de stockage externe à partir de votre ordinateur et ouvrez le Caméra de route >Ruisseau routier dossier.

  • Incidents: Déclenché par un certain événement de conduite tel qu’une collision ou un freinage brusque. Le véhicule enregistre automatiquement les vidéos de l’incident sur le disque dur embarqué.
  • Caméra de mouvement : Déclenché lorsque la caméra de mouvement détecte un mouvement lorsque vous êtes loin de votre véhicule ou déclenché par l’alarme si la caméra de mouvement est allumée. Le véhicule enregistre automatiquement les vidéos Motion Cam sur le disque dur embarqué.

  • Joué: Répertorie les vidéos que vous avez sélectionnées pour apparaître dans cet onglet vidéo de l’application Gear Guard. Les icônes suivantes différencient les types de vidéos :

    • Icône vidéo de clip Road Cam

    • Icône de la vidéo de l’incident

    • Icône vidéo Motion Cam

Le disque dur intégré supprime automatiquement les vidéos Motion Cam après 10 jours. Pour enregistrer des vidéos Motion Cam, choisissez Protection de l’engrenage dans le menu de l’écran central et choisissez Étoile  pour chaque vidéo que vous souhaitez enregistrer. Si le disque dur embarqué est plein ou presque plein, le véhicule peut supprimer les anciennes vidéos Motion Cam, en commençant par les vidéos les plus anciennes. Après avoir supprimé les vidéos Motion Cam, s’il n’y a toujours pas assez de place pour de nouveaux enregistrements, le véhicule n’enregistrera pas de nouvelles vidéos d’incident ou de motion Cam.

Dans l’onglet Vidéo Motion Cam, les vidéos du même emplacement sont organisées en un événement. Un marqueur bleu indique qu’un événement n’a pas encore été consulté.

Pour sélectionner plusieurs événements, choisissez Choisir puis sélectionnez les événements.

Vous avez les options suivantes :

  • Choisir Supprimer pour supprimer immédiatement les événements et les vidéos qui leur sont associées.

    Si vous supprimez une vidéo suivie d’une étoile à partir de la caméra de mouvement ou de l’alarme, vous la supprimez également de l’événement correspondant dans l’onglet vidéo de la caméra de mouvement. Si vous supprimez une vidéo Favoris qui est un clip Road Cam, vous la supprimez également du périphérique de stockage externe.

  • Choisir Annuler pour annuler vos modifications.

  • Choisir Plus  pour les éléments suivants :

    • Choisir Tout sélectionner pour sélectionner tous les événements.

    • Choisir Copier sur le périphérique de stockage pour copier les événements sur votre périphérique de stockage externe.

      Vous ne pouvez pas copier les clips Road Cam car ils sont automatiquement enregistrés sur votre périphérique de stockage externe.

Lorsque vous affichez un événement dans l’application Gear Guard, la première vidéo qui est lue est celle dont la caméra a détecté un mouvement en premier.

Pour regarder une vidéo :

  1. Choisissez l’onglet vidéo pour Caméra de route, Incidents, Caméra de mouvementou Joué.

  2. Sélectionnez une vidéo ou un événement.

  3. Une image de véhicule montre les différentes vues de la caméra disponibles via des vignettes. Sélectionnez la vidéo que vous souhaitez regarder.

La vidéo est lue automatiquement dans le lecteur vidéo.

Vous pouvez naviguer entre les événements.

Lecture, pause, retour en arrière ou transfert d’une vidéo. Vous avez les options suivantes :

  • Choisir Télécharger  pour copier la vidéo sur votre périphérique de stockage externe.

  • Choisir Étoile  pour afficher une vidéo dans l’onglet Vidéo suivie ou enregistrer une vidéo Motion Cam de la suppression automatique.

  • Choisir Supprimer  pour supprimer une vidéo que vous avez sélectionnée dans un événement ou pour supprimer toutes les vidéos d’un événement.

    La suppression d’une vidéo ou de toutes les vidéos d’un événement est définitive.
