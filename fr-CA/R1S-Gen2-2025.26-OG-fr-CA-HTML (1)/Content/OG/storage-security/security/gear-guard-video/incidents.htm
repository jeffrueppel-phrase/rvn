Incidents

Incidents est activé par défaut. Le véhicule enregistre automatiquement certains événements de conduite tels que les collisions ou les freinages durs.

Pour activer ou désactiver l'enregistrement:

  1. Choisissez Gear Guard dans le menu de l'écran central.

  2. Choisissez Gear Guard Features .

  3. Activez ou désactivez les incidents.

L'enregistrement commence 15 secondes avant que le véhicule ne détecte l'événement de conduite déterminé. Le véhicule enregistre des vidéos d'incident de 2 minutes sur le disque dur embarqué. Les vidéos enregistrées apparaissent dans l'onglet Vidéo des incidents.

  • Il n'y a aucune garantie que les caméras enregistrent toutes les images pertinentes pour un certain événement de conduite, comme une collision ou un freinage brutal.

  • Si le disque dur embarqué devient plein ou presque plein, le véhicule peut supprimer les anciennes vidéos Motion Cam en commençant par les vidéos les plus anciennes en premier. Après avoir supprimé les vidéos Motion Cam, s'il n'y a toujours pas assez de place pour de nouveaux enregistrements, le véhicule n'enregistrera pas de nouvelles vidéos Incident ou Motion Cam.

  • Le mode Expédition du véhicule désactive les incidents.
