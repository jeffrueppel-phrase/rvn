Utiliser le coffre avant

Assurez-vous qu'aucune personne ou objet ne se trouve sur la trajectoire du capot lorsqu'il s'ouvre ou se ferme, afin de réduire le risque d'écrasement ou d'autres blessures, ou de dommages au véhicule.

Ouvrez ou fermez le capot pour accéder au coffre avant en utilisant l'un des éléments suivants :

  • Clé à distance

  • Bouton sur le pare-chocs avant

  • Écran central

  • Bouton de verrouillage à l'intérieur du coffre avant

Ouvrir ou fermer le coffre avant avec la clé à distance

Appuyez deux fois sur la touche du coffre avant de la clé à distance pour ouvrir ou fermer le capot.

Pour mettre en pause l'ouverture ou la fermeture du capot, appuyez une fois sur la touche du coffre avant. Pour inverser la porte, appuyez à nouveau deux fois sur la touche du coffre avant.

Ouvrir ou fermer le capot avec le bouton du coffre avant

Suivez ces étapes pour utiliser la touche du coffre avant pour ouvrir le capot :

  1. Déverrouillez le véhicule.

  2. Appuyez une fois sur la touche du pare-chocs avant.

Avant de conduire le véhicule, fermez complètement le capot pour qu'il soit à niveau avec le véhicule. Si vous conduisez le véhicule avec le capot ouvert, le véhicule limite considérablement la vitesse maximale pour des raisons de sécurité.

Utiliser le bouton de piégeage du coffre avant

Ne montez pas dans le coffre avant. Il n'est pas destiné à retenir les personnes.

Le bouton d'immobilisation à l'intérieur du coffre avant s'illumine dans le noir pour plus de visibilité. Si une personne est coincée sous le capot, elle peut appuyer sur le bouton d'immobilisation pour ouvrir le coffre avant lorsque le véhicule est garé. Le bouton d'immobilisation ne fonctionnera pas si les piles sont mortes. Reportez-vous aux instructions sur la façon de Ouvrez le capot manuellement Pour plus de détails.

Le bouton de verrouillage ne libère pas la porte de la hotte si le véhicule est en position D (Conduite), N (Neutre) ou R (Réverse). Si une personne appuie sur le bouton d'immobilisation, une notification apparaît sur l'écran du conducteur. Préparez-vous à garer le véhicule immédiatement. Après avoir garé le véhicule, ouvrez le capot.
