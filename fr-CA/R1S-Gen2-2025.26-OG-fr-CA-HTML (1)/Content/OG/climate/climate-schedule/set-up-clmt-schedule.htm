Configurer un calendrier climatique

Pour configurer un Planning climatique:

  1. Choisissez Fan dans le menu de l'écran central.

  2. Dans le panneau Climat, choisissez Planifier .

  3. Choisissez Ajouter .

  4. Choisissez Modifier pour entrer un nom pour le nouveau calendrier. Puis choisissez Enregistrer.

  5. Choisissez les jours de la semaine applicables.

  6. Choisissez l'heure de départ.

  7. Pour la cabine, vous avez les options suivantes :

    • Choisissez Diminuer la température .

    • Choisissez Augmenter la température .

    • Choisissez Dégivrer pour dégivrer le pare-brise avant, la vitre arrière et les rétroviseurs latéraux.

  8. Pour les surfaces, vous avez les options suivantes :

    • Choisissez Chaleur ou Aérer . Seuls les sièges du conducteur et du passager avant peuvent être ventilés.

    • Sélectionnez les sièges que vous souhaitez chauffer ou ventiler. Choisissez Siège chauffant ou Siège aéré .

    • Choisissez Volant chauffant pour chauffer le volant pendant un temps limité.

  9. L'utilisation de l'énergie est désactivée par défaut. Si vous activez l'utilisation de l'énergie, le véhicule utilise l'énergie du chargeur, si disponible, pour maintenir l'autonomie.

  10. Choisissez Enregistrer.
