Volant chauffant

Le volant offre une option de chauffage. Pour allumer le chauffage, utilisez l’une des options suivantes sur l’écran central :

  • Choisir Climat du siège Et puis Volant chauffant .

  • Choisir Éventail  Et puis Volant chauffant dans le panneau Commandes de climatisation.

  • Choisir Éventail  puis le volant dans le panneau de commandes de ventilation.

Volant chauffant

Chauffez le volant pendant une durée limitée.

Pour chauffer le volant avant de monter dans le véhicule, utilisez l’application mobile Rivian.

  • Le chauffage s’éteint automatiquement après un certain temps.

  • Le volant chauffant dispose d’une commande séparée. Si vous désactivez l’alimentation climatique, cela n’affecte pas le volant chauffant.
