Se connecter au Wi-Fi

Vous pouvez vous connecter à des réseaux connus à tout moment. Pour ajouter un nouveau réseau, le véhicule doit être en P (Park).

  1. Recherchez les réseaux Wi‐Fi MD disponibles via l'un des outils suivants :

    • Choisissez l'icône Wi-Fi dans la barre d'état sur l'affichage central.

    • Accédez à Paramètres > Se connecter > Wi-Fi sur l'écran central.

  2. Lorsque le véhicule affiche une liste des réseaux Wi-Fi disponibles, choisissez le réseau que vous souhaitez rejoindre.

  3. Si le réseau nécessite un mot de passe, saisissez-le à l'aide du clavier qui apparaît sur l'écran central.

    Votre véhicule enregistre automatiquement le mot de passe afin que vous n'ayez pas à le saisir à nouveau.

  4. Si le réseau souhaité n'est pas répertorié, accédez aux paramètres Wi-Fi.
