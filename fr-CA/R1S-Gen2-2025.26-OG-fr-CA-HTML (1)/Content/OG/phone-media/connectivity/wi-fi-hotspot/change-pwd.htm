Changer le mot de passe

  1. Accédez aux paramètres du point d’accès Wi-Fi® via l’une des options suivantes :

    • Choisissez l'icône du point d'accès Wi-Fi puis choisissez Paramètres .

    • Accédez aux paramètres > Connecter > Point d'accès.

  2. Choisissez le nom et le mot de passe du point d'accès.

  3. Choisissez Modifier le mot de passe.

  4. Entrez le nouveau mot de passe.

    Choisissez l'icône Afficher pour voir le mot de passe.

  5. Choisissez Enregistrer.
