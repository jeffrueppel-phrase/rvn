Utilisez le point d'accès Wi-Fi

Contrôlez le point d'accès Wi-Fi® et accédez à ses paramètres via les éléments suivants :

  • Icône du point d'accès Wi-Fi dans la barre d'état sur l'écran central

  • Application Paramètres sur l'écran central

L'icône du point d'accès Wi-Fi indique l'état du point d'accès Wi-Fi.

Icon

Description

Le point d'accès Wi-Fi est activé.

Le point d'accès Wi-Fi est désactivé.

  1. Choisissez l'icône Point d'accès Wi-Fi , ou allez à Paramètres > Connecter > Point d'accès.

  2. Activez ou désactivez Point d'accès.

Si le point d'accès est activé, consultez le mot de passe, le nom du point d'accès et le nombre d'appareils connectés sur l'écran central. Sur l'appareil que vous souhaitez connecter au point d'accès Wi-Fi, suivez ces instructions :

  1. Allez dans les paramètres Wi-Fi.

  2. Sélectionnez le nom du point d'accès.

  3. Entrez le mot de passe affiché sur l'écran central.

  4. Choisissez Rejoindre.

  • Si vos appareils perdent la connexion au point d'accès Wi-Fi, vérifiez l'affichage central pour plus d'informations.

  • Le véhicule désactive automatiquement le point d'accès Wi-Fi lorsque vous quittez le véhicule.

  • Le point d'accès Wi-Fi est disponible uniquement lorsque le véhicule a une connectivité LTE. Si le véhicule perd la connectivité LTE, il désactive automatiquement le point d'accès Wi-Fi.
