Spotify

Vous devez vous connecter la première fois que vous utilisez l'application.

  1. Choisissez Média dans le menu à partir de l'écran central.

  2. Choisissez Spotify.

  3. Ouvrez votre compte sur votre téléphone ou tout autre appareil.

    Votre véhicule doit être en P (stationnement) et avoir une connectivité cellulaire ou Wi-Fi.

  4. Scannez le code QR pour vous connecter.

File d'attente Spotify

Afficher la file d'attente à l'aide de l'icône File d'attente . Pour ajouter des pistes à la file d'attente, ouvrez une liste de lecture ou un album et choisissez Ajouter à la file d'attente à côté d'une piste. Vous pouvez supprimer des pistes de la file d'attente.

Qualité de la diffusion en continu

  1. Choisissez la qualité de diffusion de musique en continu.

  2. Choisissez parmi cinq niveaux de diffusion en continu :

    • Automatique

    • Faible

    • Normal

    • Élevé

    • Très élevé
