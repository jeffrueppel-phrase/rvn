Audible

Utilisez Audible pour écouter vos livres audio préférés. Avec le son premium Rivian, certains titres sont disponibles avec l'audio spatial Dolby Atmos®, pour une expérience d'écoute améliorée.

Configuration

  1. Assurez-vous que le véhicule est en P (Stationnement).

  2. Choisissez Média dans le menu de l'affichage central.

  3. Choisissez Audible .

  4. Sélectionnez le bouton Se connecter.

  5. Scannez le code QR avec votre appareil pour vous connecter à votre compte Audible.

Écouter

  1. Choisissez Audible dans l'application Média sur l'affichage central lorsque cela est sûr de le faire.

  2. Choisissez un livre audio ou un podcast sur la page d'accueil.

  3. Votre sélection s'affiche dans le lecteur multimédia sur le côté droit de l'écran.

  4. Sélectionnez Continuer à écouter pour afficher les livres audio ou les podcasts en cours.

    5. Les podcasts ne s'affichent que si vous les avez enregistrés précédemment sur votre application mobile Audible ou sur un navigateur web.

  5. Lorsque vous sélectionnez un livre audio, vous pouvez voir une liste de tous les chapitres que vous pouvez écouter, montrant le chapitre actuel que vous écoutez.

Vous ne pouvez accéder aux paramètres et vous connecter que lorsque le véhicule est en stationnement.
