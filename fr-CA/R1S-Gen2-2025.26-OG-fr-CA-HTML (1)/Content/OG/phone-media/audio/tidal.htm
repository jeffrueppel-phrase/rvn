MARÉE

Vous pouvez écouter Tidal dans votre véhicule.

Avec Rivian Premium Audio, Spatial Audio avec Dolby Atmos® offre une expérience d'écoute améliorée lorsque du contenu Dolby Atmos est disponible.

Installation

Vous devez vous connecter à un compte payant la première fois que vous utilisez l'application.

  1. Choisissez les médias dans le menu de l'écran central.

  2. Choisissez TIDAL.

  3. Ouvrez votre compte sur votre téléphone ou autre appareil.

    Votre véhicule doit être en P (Stationnement) et disposer d'une connectivité cellulaire ou Wi-Fi.

  4. Saisissez le code sur votre appareil pour vous connecter.

File d'attente TIDAL

Afficher la file d'attente avec le Icône defile d'attente .

Qualité du streaming

Choisissez parmi quatre niveaux de streaming :

  • Normal

  • Haut

  • HiFi

  • Maître

  • Les modifications prennent effet à partir de la piste suivante jouée.

  • Le contenu téléchargé est lu dans la qualité d'origine.
