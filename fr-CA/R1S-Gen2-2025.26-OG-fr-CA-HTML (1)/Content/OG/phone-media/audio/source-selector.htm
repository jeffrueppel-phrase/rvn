Sélecteur de source

Dans l'application Media, choisissez une source média à gauche. Si vous souhaitez encore plus de sources à choisir, ouvrez le Sélecteur de source pour voir et sélectionner parmi toutes les options disponibles. Vous pouvez déplacer des sources dans et hors de votre liste de favoris.

  1. Sélectionnez l'icône Plus pour développer le panneau de gauche.

  2. La vue développée montre vos sources favorites actuelles sous Favoris à gauche et les sources restantes dans la colonne Plus à droite.

  3. Sélectionnez une source dans l'une ou l'autre colonne.

  4. Pour déplacer une source entre les colonnes, appuyez et maintenez votre sélection, puis faites-la glisser vers l'autre colonne.

  5. La colonne Favoris peut contenir un maximum de cinq sources favorites. Il y a aussi un emplacement dynamique que vous pouvez ouvrir à partir de la section Plus.

  6. Sélectionnez X pour fermer le panneau développé.
