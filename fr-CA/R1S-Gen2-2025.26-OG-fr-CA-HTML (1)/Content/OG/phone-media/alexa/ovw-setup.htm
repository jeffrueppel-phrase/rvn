Présentation et configuration

  • Alexa n'est pas pris en charge dans la province de Québec.

  • Alexa n'est disponible qu'en anglais.

Activez Alexa pour utiliser votre voix pour contrôler presque tout ce qui est géré par l'écran central. Alexa prend en charge les commandes de base du véhicule et les médias, même si vous n'avez pas de compte Amazon. Choisissez l'option Essayer Alexa sur l'écran central lors de l'embarquement du véhicule et acceptez les conditions générales pour tester le mode d'aperçu.

Se connecter

Pour une expérience entièrement personnalisée et davantage de fonctionnalités Alexa, connectez-vous à votre compte Amazon. Sur l’écran central, accédez à Paramètres > Applications > Alexa.

Votre voiture doit être garée et connectée à Internet pour pouvoir effectuer le processus de connexion.

Une fois connecté, vous pouvez effectuer les opérations suivantes en utilisant Alexa :

  • Accédez à votre calendrier.

  • Appeler les contacts par leur nom.

  • Communiquez avec n'importe quel appareil domestique Alexa.

Appelez Alexa en prononçant le mot d'activation « Alexa ». Une barre vocale chromée apparaît sous la barre d’état sur l’écran central lorsqu’Alexa est active pour indiquer quand Alexa écoute, pense ou parle. Voice Chrome est un indicateur visuel du système d'attention d'Alexa.

Dites « Alexa, arrête » à tout moment pour annuler une commande. Vous pouvez également appuyer sur la barre vocale Chrome d'Alexa pour mettre fin immédiatement à votre interaction avec Alexa.

  • Vous pouvez vous déconnecter de votre compte Amazon à tout moment via le menu Paramètres sur l'écran central.

  • Visitez les sites Web d'Amazon pour en savoir plus sur la façon dont Alexa utilise vos informations personnelles :

    • www.amazon.com/alexaterms

    • www.amazon.com/privacy

Se désengager

Vous pouvez choisir de ne pas utiliser Alexa en restant déconnecté et en n'activant pas le mode aperçu via Essayer Alexa.
