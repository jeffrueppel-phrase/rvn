Couplage Bluetooth

Votre véhicule utilise Bluetooth® pour vous connecter aux téléphones.

Se connecter à un téléphone via Bluetooth

Pour jumeler votre téléphone avec votre véhicule :

  1. Choisir Bluetooth pour permettre au véhicule de rechercher les téléphones disponibles.

  2. Choisir Ajouter pour coupler un nouveau téléphone via Bluetooth.

  3. Sur votre téléphone, allez dans les paramètres Bluetooth.

  4. Sur l’écran central, choisissez Prochain.

  5. Lorsque l’écran central répertorie votre téléphone, choisissez votre téléphone.

  6. Vérifiez que le code de l’appareil sur votre téléphone correspond au code de l’appareil sur l’écran central.

  7. Sur votre téléphone, choisissez Paire.

Après l’appairage, le véhicule enregistre sa connexion à votre téléphone. Vous n’avez pas besoin de coupler à nouveau votre téléphone.

Paramètres Bluetooth

Accédez aux paramètres Bluetooth via les éléments suivants :

  • Bluetooth

  • Média, Téléphoneet Paramètres applications à partir du menu de l’écran central

Si vous avez un téléphone enregistré que vous souhaitez supprimer, choisissez Plus puis choisissez Oublier l’appareil.
