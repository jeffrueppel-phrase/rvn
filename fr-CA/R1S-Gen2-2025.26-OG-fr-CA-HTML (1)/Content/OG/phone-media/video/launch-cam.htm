Lancez Cam

Si vous souhaitez enregistrer des événements de lancement en Mode de lancement, vous pouvez le définir et visionner les vidéos dans l'application Vidéo.

  1. Pour activer ou désactiver Launch Cam, allez à Vidéo > Lancez Cam.

  2. Sélectionnez Enregistrer la vidéo pendant les événements de lancement pour activer Launch Cam.

  3. Sélectionnez Configurer maintenant.

    4. Vous devez avoir un périphérique de stockage externe pour utiliser Launch Cam. Ce n'est pas nécessaire si vous avez déjà ajouté un périphérique de stockage pour Road Cam.

  4. Vous pouvez sélectionner Aller aux modes de conduite pour utiliser le Mode de lancement ou d'autres modes de conduite.
