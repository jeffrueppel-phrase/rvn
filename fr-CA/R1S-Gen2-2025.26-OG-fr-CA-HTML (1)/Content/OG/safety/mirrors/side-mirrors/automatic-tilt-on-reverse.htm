Rétroviseur latéral Auto-Tilt in Reverse

Utilisez l'inclinaison automatique du rétroviseur latéral pour offrir une meilleure vue latérale des alentours et des bordures de trottoir lorsque vous stationnez le véhicule en parallèle ou que vous le reculez dans un endroit. Cette fonctionnalité est désactivée par défaut. Pour lui permettre:

  1. Accédez à Paramètres > Véhicule > Accès sur l'écran central.

  2. Activez le rétroviseur latéral basculant automatiquement.

La fonction activée, les rétroviseurs latéraux conducteur et passager basculent lorsque vous passez le véhicule en R (marche arrière). Une fois que vous avez garé le véhicule et que vous l'avez décalé de la marche arrière, les rétroviseurs latéraux reviennent à leur position de conduite normale.

Personnalisez les positions d'inclinaison automatique du rétroviseur latéral

  1. Accédez à Paramètres > Véhicule > Accès sur l'écran central.

  2. Sous Rétroviseur latéral inclinable, choisissez Personnaliser. Les miroirs se déplacent dans la position d'inclinaison d'usine par défaut, ou si vous avez personnalisé dans le passé, dans la dernière position d'inclinaison enregistrée.

  3. Utilisez les commandes du pouce sur le volant pour régler les rétroviseurs.

  4. Choisissez N'oubliez pas d'enregistrer les nouveaux paramètres dans votre profil de pilote.

Pour remettre les paramètres d'inclinaison automatique du rétroviseur latéral à la valeur par défaut d'usine, choisissez Restaurer.

Annuler Rétroviseur latéral Inclinaison automatique en marche arrière

Pour annuler l'inclinaison automatique du rétroviseur latéral lorsque le véhicule est en marche arrière, appuyez sur le bouton gauche du pouce situé sur le côté droit du volant. Si vous devez l'annuler lorsque vous êtes près d'un garage associé au véhicule:

  1. Faites défiler la molette sur le côté droit du volant vers le bas jusqu'à ce que vous voyiez l'indicateur de rétroviseur latéral sur l'écran conducteur.

  2. Appuyez sur le bouton gauche du pouce sur le côté droit du volant pour annuler.
