Ajustez les rétroviseurs latéraux

  1. Allez à votre profil de conducteur sur l'écran central.

  2. Choisissez Rétroviseurs latéraux.

  3. Utilisez les commandes et les boutons sur le côté gauche du volant pour ajuster le rétroviseur côté conducteur. Utilisez les commandes sur le côté droit du volant pour ajuster le rétroviseur côté passager.

  4. Choisissez Se souvenir pour enregistrer les paramètres. Après avoir enregistré les paramètres dans votre profil de conducteur, les rétroviseurs latéraux s'ajusteront automatiquement lorsque vous déverrouillerez le véhicule de l'extérieur.
