Verrouiller ou déverrouiller les fenêtres des passagers

Pour verrouiller ou déverrouiller les deux fenêtres arrière des passagers :

  1. Choisissez le Véhicule bouton dans la barre d'état supérieure de l'affichage central.

  2. Dans le panneau d'accès, choisissez le Raccourcis bouton dans le coin supérieur droit.

  3. Choisissez Verrouillages de fenêtres.

    1. Le réglage verrouille les deux fenêtres arrière des passagers (par défaut).

    2. L'INDICATEUR ON : L'indicateur ARRIÈRE apparaît.

    3. Les Verrouillages de fenêtres sont mis en surbrillance pour indiquer que le réglage est ACTIVÉ.

Pour personnaliser le réglage et verrouiller une ou plusieurs fenêtres de passagers, y compris la fenêtre du passager avant :

  1. Choisissez le Modifier bouton à droite de Verrouillages de fenêtres dans le menu Raccourcis.

  2. Sur l'image du véhicule dans le panneau Modifier les Verrouillages de fenêtres, sélectionnez la ou les fenêtres de passagers à verrouiller ou déverrouiller.

    • Une icône de verrou ouvert indique que le verrou de la fenêtre des passagers est réglé sur OFF.

    • Une icône de verrou fermé indique que le verrou de la fenêtre des passagers est réglé sur ON.

  3. Choisissez Terminé.

    1. Le réglage verrouille les fenêtres passagers sélectionnées.

    2. Si toutes les fenêtres sont réglées pour se verrouiller, le ON : L'indicateur TOUT PASSAGER apparaît dans les Verrouillages de fenêtres du menu Raccourcis.

    3. Si l'une des trois fenêtres, ou une fenêtre arrière et la fenêtre avant sont réglées pour se verrouiller, le ON : L'indicateur PERSONNALISÉ apparaît dans les Verrouillages de fenêtres du menu Raccourcis.

Pour désactiver le verrouillage des fenêtres passagers, choisissez Verrouillages de fenêtres. Le bouton Verrouillages de fenêtres se désélectionne pour indiquer que le réglage est OFF.

  • Le verrouillage d'une fenêtre passager désactive l'interrupteur de la fenêtre sur la porte passager. Cela ne désactive pas l'interrupteur pour cette fenêtre sur la porte du conducteur.

  • Les Verrouillages de fenêtres se souviennent du réglage précédent au démarrage du véhicule.

  • Pour effacer les réglages personnalisés, dans le panneau Modifier les Verrouillages de fenêtres, désélectionnez toutes les fenêtres. La prochaine fois que vous activez les Verrouillages de fenêtres, le réglage revient à la valeur par défaut, verrouillant les deux fenêtres passagers arrière.
