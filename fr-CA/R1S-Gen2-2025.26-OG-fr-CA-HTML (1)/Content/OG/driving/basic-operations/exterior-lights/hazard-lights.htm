Feux de détresse

Appuyez sur le bouton Feux de détresse sur la console de bord pour allumer les feux de détresse. Appuyez sur le bouton encore une fois pour éteindre les feux de détresse. Voir SOS d'urgence.

Les clignotants ont priorité sur les feux de détresse. Si les feux de détresse sont allumés et que vous poussez la tige des feux vers le haut ou vers le bas pour indiquer un changement de virage ou de voie, les clignotants clignotent et les feux de détresse s'éteignent temporairement. Une fois les clignotants arrêtés, les feux de détresse reprennent.
