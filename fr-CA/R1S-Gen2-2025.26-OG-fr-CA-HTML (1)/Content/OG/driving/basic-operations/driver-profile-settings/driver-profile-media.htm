Application média préférée

Le véhicule enregistre les applications média utilisées en dernier dans le profil du conducteur. Lorsqu'il reconnaît un Rivian ID au début d'un trajet, l'application média enregistrée du trajet précédent se charge automatiquement.

Pour les abonnés Connect+ , les applications enregistrées incluent Spotify®, Apple Music, Audible, TIDAL®, et TuneIn®.
