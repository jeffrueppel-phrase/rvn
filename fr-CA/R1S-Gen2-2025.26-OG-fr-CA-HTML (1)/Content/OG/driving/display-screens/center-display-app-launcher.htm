Lanceur d'applications

Le lanceur d’applications est situé en bas au centre de l’écran central.

  • Choisissez n’importe quelle application dans la barre d’état ou dans le menu développé pour la lancer.

  • Faites glisser la poignée de la barre d'applications vers le haut ou vers le bas pour afficher ou masquer le menu d'applications développé. Vous pouvez également appuyer sur la zone située au-dessus du menu de l’application développé pour le réduire.

Pour accéder rapidement à vos favoris, vous pouvez personnaliser les applications qui apparaissent dans la barre d'état système et dans quel ordre, en procédant comme suit :

  • Appuyez longuement sur un favori dans la barre d'applications et faites-le glisser vers un autre emplacement pour le réorganiser.

  • Dans le menu d'applications développé, maintenez une application enfoncée et faites-la glisser vers l'un des emplacements de la barre d'applications pour remplacer un favori existant.

Avec le menu d'applications développé exposé, vous pouvez faire glisser une application favorite de la barre d'état système vers le menu développé pour réduire le nombre d'emplacements favoris.

  • Le véhicule doit être en P (Park) pour que vous puissiez modifier l'ordre de l'application.

  • L'application la plus à droite dans la barre d'état est l'application que vous avez utilisée le plus récemment et qui n'est pas déjà enregistrée comme favorite.

  • Les nouvelles applications apparaissent dans le menu des applications développé avec un point à côté d'elles.
