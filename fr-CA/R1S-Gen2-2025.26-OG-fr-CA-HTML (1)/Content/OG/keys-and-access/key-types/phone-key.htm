Clé téléphonique

Pour verrouiller ou déverrouiller le véhicule à l'aide de votre téléphone, utilisez l'une des méthodes suivantes :

  • Sélectionnez le Verrouiller ou Déverrouiller dans l'application mobile Rivian.

  • Utilisez le verrouillage par proximité lorsque vous êtes près du véhicule.

Vous pouvez associer jusqu'à quatre Bluetooth® Smartphones à faible consommation d'énergie avec le véhicule. Voir Application mobile Rivian pour les instructions.
