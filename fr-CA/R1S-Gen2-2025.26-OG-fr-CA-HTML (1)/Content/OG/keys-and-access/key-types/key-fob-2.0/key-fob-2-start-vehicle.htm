Démarrez le véhicule avec une télécommande

Pour conduire le véhicule sans clé de téléphone, démarrez-le dans les 60 secondes suivant le déverrouillage du véhicule avec la télécommande. Si 60 secondes s'écoulent avant que vous ne démarriez le véhicule, touchez la télécommande au plateau de chargement sur le dessus de la console centrale pendant quelques secondes. Le véhicule émet un signal pour confirmer. Une fois que vous entendez le signal, vous pouvez démarrer le véhicule.

Si la batterie de la télécommande est morte, placez la télécommande sur le plateau de chargement (côté bouton vers le haut) jusqu'à ce que vous entendiez le signal.
