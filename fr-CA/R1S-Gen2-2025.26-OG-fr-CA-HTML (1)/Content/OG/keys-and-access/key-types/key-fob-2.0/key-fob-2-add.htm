Ajouter un porte-clés

Pour ajouter un porte-clés et le coupler avec votre véhicule, scannez le code QR sur la boîte contenant le porte-clés, ou:

  1. Ouvrez l'application mobile Rivian sur votre téléphone.

  2. Allez dans Drivers et clés et choisissez Add Key Fob pour lancer le processus d'appariement.
