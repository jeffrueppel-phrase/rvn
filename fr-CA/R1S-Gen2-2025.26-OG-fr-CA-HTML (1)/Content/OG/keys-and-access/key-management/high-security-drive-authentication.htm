Lecteur multifacteur

La conduite multifacteur nécessite l'approbation du propriétaire clé avant le démarrage du véhicule. Pour activer ce paramètre :

  1. Connectez-vous au compte du propriétaire principal du véhicule dans l'application mobile Rivian.

  2. Accédez à Sécurité et accès et activez le lecteur multifacteur.

Pour démarrer le véhicule lorsque l'entraînement multifacteur est activé :

  1. Déverrouillez et entrez dans le véhicule.

  2. Appuyez sur le frein. Une notification s'affiche sur l'écran du conducteur et une demande d'approbation du disque apparaît sur le téléphone du propriétaire de la clé.

  3. Le propriétaire de la clé approuve le lecteur.

  4. Une fois la notification affichée au conducteur effacée, démarrez le véhicule.

Si le propriétaire de la clé ne reçoit pas la demande d'approbation sur son téléphone ou si l'avis sur l'écran du conducteur n'est pas clair, effectuez l'une des opérations suivantes dans la notification affichée par le conducteur :

  • Choisissez Envoyer à nouveau pour renvoyer la demande au téléphone.

  • Cliquez sur Essayer d'une autre façon. Cela ouvre un clavier sur l'écran du conducteur qui demande un code d'accès rapide. Si vous ne connaissez pas le code et que vous êtes le propriétaire de la clé :

    1. Choisissez Où est mon mot de passe sous le clavier.

    2. Scannez le code QR avec votre téléphone. Le code d'accès s'affiche sur votre téléphone.

    3. Saisissez le mot de passe dans le clavier.

Pour désactiver le lecteur multifacteur, accédez à Sécurité et accès dans l'application mobile Rivian.

  • Seul le propriétaire principal peut activer et désactiver le lecteur multifacteur. Le réglage s'applique à tous les conducteurs et aux clés associés au véhicule.

  • Pour approuver une conduite, le propriétaire de la clé doit avoir un compte Rivian associé à la clé, et la clé doit être jumelée au véhicule.

  • Le propriétaire de la clé peut également accéder au code secret à partir de l'écran Sécurité et accès de l'application mobile Rivian. Choisissez Afficher le code secret basé sur l'heure.
