Renommer un garage

  1. Choisissez le bouton Garages . dans la barre d'état en haut de l'écran central.

  2. Choisissez le Paramètres bouton pour ouvrir la page Garages.

  3. Choisir > à droite du nom du garage.

  4. Sélectionnez le Modifier bouton .

  5. Utilisez le clavier pour entrer un nouveau nom pour le garage.

  6. Choisissez Enregistrer.
