Verrouillage des portes pour enfants

Le verrouillage des portes pour enfants est disponible sur une ou les deux portes arrière du véhicule. Lorsqu'il est activé, le verrouillage des portes pour enfants empêche les passagers d'ouvrir la porte depuis l'intérieur du véhicule. Avec le verrouillage des portes pour enfants activé, vous ne pouvez ouvrir les portes arrière que depuis l'extérieur.

Pour activer ou désactiver le verrouillage des portes pour enfants :

  1. Dans le menu Contrôles rapides Raccourcis , allez à Verrouillage des portes enfant.

  2. Choisissez l'option Modifier bouton.

  3. Sélectionnez une ou les deux portes pour activer ou désactiver le verrouillage des portes pour enfants.

  4. Choisissez Terminé.
