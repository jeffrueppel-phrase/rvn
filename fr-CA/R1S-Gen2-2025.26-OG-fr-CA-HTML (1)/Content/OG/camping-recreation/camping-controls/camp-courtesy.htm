Politesse de camping

Pour se déplacer dans le véhicule sans déranger les autres, activez la courtoisie de camping pour désactiver ce qui suit :

  • Les feux extérieurs, tels que les feux qui s'allument automatiquement lorsque vous verrouillez ou déverrouillez le véhicule ou ouvrez les portes

  • Les sons du véhicule, tels que les sons de verrouillage et les carillons extérieurs

  • Le verrouillage ou le déverrouillage par proximité

  • Le son d'alarme Gear Guard

Lorsque la politesse de camping est activée, utilisez la commande des feux pour allumer ou éteindre les phares.

Le véhicule règle les feux sur le mode automatique dans les conditions suivantes :

  • Vous désactivez la politesse de camping.

  • Vous conduisez plus vite que la vitesse du .

  • Le véhicule se réveille du sommeil, et vous déplacez le véhicule hors de P (Parc).

Lorsque la fonction Camp Courtesy est active, le véhicule peut limiter le chauffage ou la climatisation pour réduire le bruit ; cependant, vous pouvez ajuster manuellement les commandes de climatisation pour accélérer le chauffage ou le refroidissement.
