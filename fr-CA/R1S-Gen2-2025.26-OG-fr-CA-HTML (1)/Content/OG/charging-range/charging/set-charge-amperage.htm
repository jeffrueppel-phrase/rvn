Définir l'ampérage de charge

Pour réduire le niveau de courant que le véhicule consomme pendant la charge, modifiez le réglage de l'ampérage.

  1. Aller à Énergie > Recharge > Ampérage.

  2. Sélectionnez la valeur d'ampères appropriée.

    • Si vous réglez l'intensité de charge lorsque vous fixer un calendrier de facturation, le véhicule enregistre le paramètre pour les futures sessions de charge programmées.

    • Lorsque le véhicule termine sa session de charge, le réglage de l'ampérage est réinitialisé à la valeur maximale si vous n'avez pas défini de planning de charge ou si vous êtes éloigné de l'emplacement enregistré.
