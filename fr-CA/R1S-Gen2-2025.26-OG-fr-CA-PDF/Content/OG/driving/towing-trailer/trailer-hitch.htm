Accroc

Le R1S dispose d’un Classe IV récepteur et un connecteur de remorque à 7 broches situé derrière le couvercle de l’attelage de remorque sous le pare-chocs arrière.

Pour retirer le couvercle de l’attelage de remorque :

  1. À l’aide d’un tournevis à tête plate, desserrez les vis du couvercle de l’attelage de remorque d’un quart de tour dans le sens inverse des aiguilles d’une montre.

    2. Pour éviter la casse, ne relâchez pas complètement les vis. Les vis restent fixées au couvercle de l’attelage de remorque.

  2. Saisissez le couvercle et tirez-le vers le bas verticalement pour le retirer.

Pour réinstaller le couvercle de l’attelage de remorque :

  1. Alignez les languettes du couvercle de l’attelage de remorque avec la fente de l’attelage de remorque. Assurez-vous que le couvercle est vertical.

  2. Soulevez le couvercle de l’attelage de remorque verticalement pour insérer les languettes dans le véhicule. Poussez le couvercle dans la fente de l’attelage de remorque jusqu’à ce qu’il affleure le pare-chocs.

  3. À l’aide d’un tournevis à tête plate, tournez les deux vis d’un quart de tour dans le sens des aiguilles d’une montre. Arrêtez de serrer lorsque les rainures sont parallèles au bord inférieur du couvercle de l’attelage.

    Ne serrez pas trop les vis.
