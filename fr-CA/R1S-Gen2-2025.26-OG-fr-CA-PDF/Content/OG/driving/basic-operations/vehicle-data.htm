Données du véhicule

Pour naviguer vers À propos de votre véhicule, allez à Paramètres > À propos sur l'affichage central.

À partir de À propos de votre véhicule, vous pouvez faire ce qui suit :

  • Nommez votre véhicule.

  • Voir le compteur kilométrique.

  • Voir le numéro d'identification du véhicule (NIV).

  • Voir le forfait du véhicule.

  • Voir des informations telles que le pack de batteries.

  • Contactez Rivian.

Nommez votre véhicule

Personnalisez le nom de votre véhicule sur l'affichage central. Allez à Paramètres > À propos et sélectionnez le champ du nom du véhicule. Vous pouvez également nommer votre véhicule dans l'application Rivian.

Si vous souhaitez ajouter des emojis à votre nom de véhicule, utilisez l'application Rivian.

Données de voyage

Stockez les données de deux voyages sous Voyage A et Voyage B. Chaque voyage suit les informations suivantes :

  • Distance totale

  • Vitesse moyenne

  • Efficacité

  • Durée

  • Énergie totale

Pour réinitialiser vos données de voyage :

  1. Allez à Paramètres > À propos sur l'affichage central.

  2. Sélectionnez le voyage que vous souhaitez réinitialiser.

  3. Sélectionnez Réinitialiser le voyage.
