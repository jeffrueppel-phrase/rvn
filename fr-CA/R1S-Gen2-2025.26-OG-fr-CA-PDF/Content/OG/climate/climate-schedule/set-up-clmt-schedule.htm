Configurer un calendrier climatique

Pour configurer un Horaire du climat:

  1. Choisir Éventail dans le menu de l’écran central.

  2. Dans le panneau Climat, choisissez Horaire .

  3. Choisir Ajouter .

  4. Choisir Éditer pour entrer un nom pour la nouvelle planification. Choisissez ensuite Sauvegarder.

  5. Choisissez les jours de la semaine applicables.

  6. Choisissez l’heure de départ.

  7. Pour la cabine, vous avez les options suivantes :

    • Choisir Diminuer la température .

    • Choisir Augmenter la température .

    • Choisir Dégivrer pour dégivrer le pare-brise avant, la lunette arrière et les rétroviseurs latéraux.

  8. Pour les surfaces, vous disposez des options suivantes :

    • Choisir Chaleur ou Ventiler . Seuls les sièges conducteur et passager avant peuvent ventiler.

    • Sélectionnez les sièges que vous souhaitez chauffer ou aérer. Choisir Siège chauffant ou Siège aérateur .

    • Choisir Volant chauffant pour chauffer le volant pendant une durée limitée.

  9. La consommation d’énergie est désactivée par défaut. Si vous activez la consommation d’énergie, le véhicule utilise l’énergie du chargeur, le cas échéant, pour maintenir l’autonomie.

  10. Choisir Sauvegarder.
