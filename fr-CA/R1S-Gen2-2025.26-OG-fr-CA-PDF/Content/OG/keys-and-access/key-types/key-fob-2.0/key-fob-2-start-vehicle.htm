Démarrez le véhicule avec un porte-clés

Pour conduire le véhicule sans clé de téléphone, démarrez-le dans les 60 secondes suivant le déverrouillage du véhicule avec le porte-clés. Si 60 secondes s’écoulent avant de démarrer le véhicule, touchez le porte-clés sur le plateau de charge situé sur le dessus de la console centrale pendant quelques secondes. Le véhicule sonne pour confirmer. Une fois que vous entendez le carillon, vous pouvez démarrer le véhicule.

Si la pile du porte-clés est morte, posez-la sur le plateau de charge (côté bouton vers le haut) jusqu’à ce que vous entendiez le carillon.
