Conduite multi-facteurs

La conduite multi-facteurs nécessite l'approbation du propriétaire de la clé avant le démarrage du véhicule. Pour activer ce paramètre :

  1. Connectez-vous au compte principal du propriétaire du véhicule dans l'application mobile Rivian.

  2. Allez à Sécurité et accès et activez Conduite multi-facteurs.

Pour démarrer le véhicule lorsque la conduite multi-facteurs est activée :

  1. Déverrouillez et entrez dans le véhicule.

  2. Appuyez sur le frein. Une notification apparaît sur l'affichage du conducteur, et une demande d'approbation de la conduite apparaît sur le téléphone du propriétaire de la clé.

  3. Le propriétaire de la clé approuve la conduite.

  4. Après que la notification de l'affichage du conducteur se soit effacée, démarrez le véhicule.

Si le propriétaire de la clé ne reçoit pas la demande d'approbation sur son téléphone, ou si la notification sur l'affichage du conducteur ne s'efface pas, faites l'une des choses suivantes dans la notification de l'affichage du conducteur :

  • Choisissez Renvoyer pour renvoyer la demande au téléphone.

  • Choisissez Essayer une autre méthode. Cela ouvre un clavier sur l'affichage du conducteur demandant un code d'accès sensible au temps. Si vous ne connaissez pas le code, et que vous êtes le propriétaire de la clé :

    1. Choisissez Où est mon code d'accès sous le clavier.

    2. Scannez le code QR avec votre téléphone. Le code d'accès apparaît sur votre téléphone.

    3. Entrez le code d'accès dans le clavier.

Pour désactiver Multi-factor Drive, allez à Sécurité et accès dans l'application mobile Rivian.

  • Seul le propriétaire principal peut activer et désactiver Multi-factor Drive. Le paramètre s'applique à tous les conducteurs et clés associés au véhicule.

  • Pour approuver un trajet, le propriétaire de la clé doit avoir un compte Rivian associé à la clé, et la clé doit être appairée avec le véhicule.

  • Le propriétaire de la clé peut également accéder au code d'accès depuis l'écran Sécurité et accès dans l'application mobile Rivian. Choisissez Voir le code d'accès basé sur le temps.
