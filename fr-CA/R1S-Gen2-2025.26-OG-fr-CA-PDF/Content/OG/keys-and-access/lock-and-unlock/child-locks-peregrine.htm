Verrous pour enfants

Les verrous pour enfants sont disponibles sur l'une ou l'autre des portes arrière du véhicule. Lorsqu'ils sont engagés, les verrous pour enfants empêchent les passagers d'ouvrir la porte de l'intérieur du véhicule. Avec les verrous pour enfants en place, vous ne pouvez ouvrir les portes arrière du véhicule que de l'extérieur.

Pour engager ou désengager les verrous pour enfants :

  1. Dans le menu Contrôles rapides Raccourcis, allez à Verrous de porte pour enfants.

  2. Choisissez le Modifier bouton.

  3. Sélectionnez une ou les deux portes pour activer ou désactiver les verrous pour enfants.

  4. Choisissez Terminé.
