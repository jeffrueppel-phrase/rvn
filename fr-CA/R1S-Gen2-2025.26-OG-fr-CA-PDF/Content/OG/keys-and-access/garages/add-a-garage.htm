Ajouter un garage

Utilisez la fonction Ajouter un garage avec votre télécommande de porte de garage pour coupler un ouvre-porte de garage installé avec le véhicule et, éventuellement, pour configurer une invite à une touche pour accéder à votre garage lorsque vous vous approchez ou quittez son emplacement. Procédez comme suit :

  1. Se garer à proximité et devant le garage (pas sous la porte).

  2. Choisissez le bouton Garages dans la barre d'état en haut de l'écran central.

  3. Choisissez le bouton (+) pour ajouter un garage.

  4. Suivez les instructions à l'écran pour ajouter le garage.

  • Pour certains appariements de garage, vous aurez peut-être besoin des éléments suivants :

    • Accès à l'unité matérielle de l'ouvre-porte de garage installée

    • Une autre personne pour vous aider

  • Vous pouvez associer jusqu'à 8 garages à un véhicule.

  • Vous pouvez également accéder au menu Garages sur l'écran central à partir de Paramètres  > Connecter > Garages.
