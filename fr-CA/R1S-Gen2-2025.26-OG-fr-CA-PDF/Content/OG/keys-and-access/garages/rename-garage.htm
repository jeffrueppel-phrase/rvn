Renommer un garage

  1. Choisissez le bouton Garages dans la barre d'état en haut de l'affichage central.

  2. Choisissez le bouton Paramètres pour ouvrir la page des garages.

  3. Choisissez > à droite du nom du garage.

  4. Sélectionnez le bouton Modifier .

  5. Utilisez le clavier pour entrer un nouveau nom pour le garage.

  6. Choisissez Enregistrer.
