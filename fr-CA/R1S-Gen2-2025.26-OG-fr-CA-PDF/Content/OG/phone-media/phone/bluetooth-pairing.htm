Appairage Bluetooth

Votre véhicule utilise la technologieBluetooth® pour se connecter aux téléphones.

Se connecter à un téléphone via Bluetooth

Pour connecter votre téléphone à votre véhicule :

  1. Choisissez Bluetooth pour permettre au véhicule de rechercher les téléphones disponibles.

  2. Choisissez Ajouter pour coupler un nouveau téléphone via Bluetooth.

  3. Sur votre téléphone, accédez aux paramètres Bluetooth.

  4. Sur l'écran central, sélectionnez Suivant.

  5. Lorsque l'écran central affiche votre téléphone, sélectionnez-le.

  6. Vérifiez que le code de l'appareil sur votre téléphone correspond au code de l'appareil affiché sur l'écran central.

  7. Sur votre téléphone, sélectionnez « Appairer ».

Après le couplage, le véhicule enregistre sa connexion à votre téléphone. Vous n'avez pas besoin de reconnecter votre téléphone.

Paramètres Bluetooth

Accédez aux paramètres Bluetooth via les éléments suivants :

  • Bluetooth

  • Applications Médias, Téléphone et Paramètres dans le menu de l'écran central

Si vous avez enregistré un téléphone que vous souhaitez supprimer, sélectionnez Plus. puis sélectionnez « Oublier l'appareil ».
