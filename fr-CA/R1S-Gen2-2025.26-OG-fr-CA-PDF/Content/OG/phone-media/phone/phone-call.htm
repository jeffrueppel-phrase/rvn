Appel téléphonique

Pour gérer les appels téléphoniques, associez votre téléphone à votre véhicule via Bluetooth®.

Pour ajuster le volume, utilisez le bouton Volume en bas de l'écran central ou faites défiler le contrôle de pouce gauche sur le volant.

Passer un appel

Choisissez Téléphone dans le menu de l'écran central. Vous pouvez passer un appel en utilisant l'une des options suivantes :

  • Favoris : Choisissez l'icône Favoris pour accéder à une liste de vos contacts favoris.

  • Appels récents : Choisissez l'icône Appels récents pour accéder à une liste de vos appels téléphoniques récents.

  • Contacts : Choisissez l'icône Contacts pour accéder à une liste de vos contacts.

  • Clavier: Choisissez l'icône Clavier pour afficher un clavier et composer un numéro de téléphone.

    Lorsque vous composez un numéro de téléphone, des contacts peuvent apparaître. Vous pouvez choisir un contact à appeler.

    Pendant un appel actif, vous pouvez choisir Ajouter un appel et composer un numéro de téléphone pour ajouter un autre appel.

  • Si Alexa est active, vous pouvez utiliser Alexa pour passer un appel.

  • Le support pour les appels en conférence ou parallèles peut varier, selon votre téléphone spécifique et votre réseau cellulaire.

Gérer les appels entrants et actifs

Pour gérer les appels téléphoniques entrants et actifs, vous avez deux méthodes disponibles :

  • Utilisez le contrôle multifonctionnel sur le côté gauche du volant. Les contrôles de gauche correspondent aux options sur le panneau qui apparaît sur l'affichage du conducteur.

  • Utilisez l'affichage central. Le haut du panneau liste les appels actifs.

Le tableau ci-dessous décrit les contrôles téléphoniques qui peuvent apparaître sur les affichages du conducteur et central.

Changer d'appels

Affichage du conducteur : Si vous avez deux appels actifs, changez entre eux.

Affichage central : Si vous avez plus d'un appel actif, choisissez entre les appels en haut du panneau. (Il n'y a pas d'icône.)

Fusionner

Affichage central : Si vous avez plus d'un appel actif, vous pouvez les fusionner.

Keypad

Affichage central : Lors d'un appel actif, choisissez l'icône Clavier pour répondre à un système téléphonique automatisé.

Pour passer un appel, choisissez Téléphone dans le menu de l'écran central. Puis choisissez l'icône Clavier et composez le numéro sur le clavier.

Faites des appels SOS ou d'urgence

Les appels SOS ou d'urgence ne nécessitent pas que le téléphone soit appairé au véhicule via Bluetooth. Voir Urgence SOS.
