Lancer Cam

Si vous souhaitez enregistrer les événements de lancement dans Mode de lancement, vous pouvez le configurer et visualiser les vidéos dans l'application Vidéo.

  1. Pour activer ou désactiver Launch Cam, allez à Vidéo > Lancer Cam.

  2. Sélectionnez Enregistrez une vidéo pendant les événements de lancement pour activer Launch Cam.

  3. Sélectionnez Configurer maintenant.

    4. Vous devez disposer d'un périphérique de stockage externe pour utiliser Launch Cam. Cela n'est pas nécessaire si vous avez déjà ajouté un périphérique de stockage pour Road Cam.

  4. Vous pouvez sélectionner Aller aux modes de conduite pour utiliser le mode de lancement ou d'autres modes de conduite.
