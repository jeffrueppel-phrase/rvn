Changer le nom

  1. Accédez aux paramètres du point d'accès Wi-Fi® via l'une des options suivantes :

    • Choisissez l'icône Point d'accès Wi-Fi et choisissez ensuite Paramètres .

    • Allez à Paramètres > Connecter > Point d'accès.

  2. Choisissez le nom et le mot de passe du point d'accès.

  3. Choisissez Renommer le point d'accès.

  4. Entrez le nouveau nom.

  5. Choisissez Enregistrer.
