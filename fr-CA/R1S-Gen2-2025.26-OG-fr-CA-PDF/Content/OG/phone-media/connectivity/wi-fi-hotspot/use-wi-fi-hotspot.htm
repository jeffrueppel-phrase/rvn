Utiliser le Wi-Fi Hotspot

Contrôlez le point d'accès Wi-Fi® et accédez à ses paramètres par les moyens suivants :

  • Icône Wi-Fi Hotspot dans la barre d'état sur l'affichage central

  • Application de réglages sur l'écran central

L'icône de point d'accès Wi-Fi indique l'état du point d'accès Wi-Fi.

Icon

Description

Le point d'accès Wi-Fi est activé.

Le point d'accès Wi-Fi est éteint.

  1. Choisissez l'icône Wi-Fi Hotspot , ou allez dans Paramètres > Connexion > Hotspot.

  2. Activez ou désactivez Hotspot.

Si le point d'accès est activé, affichez le mot de passe, le nom du point d'accès et le nombre de périphériques connectés sur l'écran central. Sur l’appareil que vous souhaitez connecter au point d’accès Wi-Fi, suivez les instructions suivantes :

  1. Accédez aux paramètres Wi-Fi.

  2. Sélectionnez le nom du hotspot.

  3. Entrez le mot de passe affiché sur l'écran central.

  4. Choisissez Joindre.

  • Si vos appareils perdent la connexion au point d'accès Wi-Fi, vérifiez l'affichage central pour plus d'informations.

  • Le véhicule coupe automatiquement le point d'accès Wi-Fi lorsque vous sortez du véhicule.

  • Le point d'accès Wi-Fi n'est disponible que lorsque le véhicule dispose d'une connectivité LTE. Si le véhicule perd la connectivité LTE, il coupe automatiquement le point d'accès Wi-Fi.
