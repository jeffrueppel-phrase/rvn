MARÉE

Vous pouvez écouter Tidal dans votre véhicule.

Avec Rivian Premium Audio, Spatial Audio avec Dolby Atmos ® offre une expérience d'écoute améliorée lorsque du contenu Dolby Atmos est disponible.

Configuration

Vous devez vous connecter à un compte payant la première fois que vous utilisez l'application.

  1. Choisissez Média dans le menu à partir de l'écran central.

  2. Choisissez TIDAL.

  3. Ouvrez votre compte sur votre téléphone ou tout autre appareil.

    Votre véhicule doit être en P (stationnement) et avoir une connectivité cellulaire ou Wi-Fi.

  4. Entrez le code sur votre appareil pour vous connecter.

File d'attente TIDAL

Afficher la file d'attente à l'aide de l'icône File d'attente .

Qualité de la diffusion en continu

Choisissez parmi quatre niveaux de diffusion en continu :

  • Normal

  • Élevé

  • HiFi

  • Capitaine

  • Les changements prennent effet à partir de la piste suivante jouée.

  • Le contenu téléchargé est diffusé à la qualité originale.
