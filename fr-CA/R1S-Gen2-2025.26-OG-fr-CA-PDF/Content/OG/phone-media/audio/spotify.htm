Spotify

Vous devez vous connecter la première fois que vous utilisez l'application.

  1. Choisissez les médias dans le menu de l'écran central.

  2. Choisissez Spotify.

  3. Ouvrez votre compte sur votre téléphone ou autre appareil.

    Votre véhicule doit être en P (Stationnement) et disposer d'une connectivité cellulaire ou Wi-Fi.

  4. Scannez le code QR pour vous connecter.

File d'attente Spotify

Afficher la file d'attente avec l'icône File d'attente . Pour ajouter des pistes à la file d'attente, ouvrez une liste de lecture ou un album et choisissez Ajouter à File d'attente à côté d'une piste. Vous pouvez supprimer des pistes de la file d'attente.

Qualité du streaming

  1. Choisissez la qualité du streaming musical.

  2. Choisissez parmi cinq niveaux de streaming :

    • Automatique

    • Faible

    • Normal

    • Haut

    • Très élevé
