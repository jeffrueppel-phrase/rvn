Apple Music

Vous pouvez écouter Apple Music dans votre véhicule de la manière habituelle que vous utilisez depuis votre appareil.

Vous devez être un abonné payant pour utiliser Apple Music dans votre véhicule.

Installation

  1. Assurez-vous que le véhicule est en position P (Park).

  2. Choisissez les médias dans le menu de l'écran central.

  3. Choisissez Apple Music .

  4. Sélectionnez le bouton Se connecter .

  5. Scannez le code QR avec votre appareil pour vous connecter à votre compte Apple.

Écouter

Choisissez Apple Music dans l'application Média sur l'écran central.

Paramètres

Le véhicule doit être en position P (Park) pour utiliser les paramètres.

  • Avec Rivian Premium Audio, Spatial Audio avec Dolby Atmos® offre une expérience d'écoute améliorée. C'est activé par défaut, mais vous pouvez y accéder Paramètres  pour le désactiver. Le contenu se charge plus rapidement lorsque l'audio spatial est désactivé.

  • Vous pouvez activer les restrictions de contenu pour supprimer le contenu explicite du menu. Aller à Paramètres  pour l'activer.
