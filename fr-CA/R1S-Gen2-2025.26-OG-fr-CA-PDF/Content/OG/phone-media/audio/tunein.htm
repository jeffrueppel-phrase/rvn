Accordez-vous

Aucune connexion n'est requise.

  1. Choisissez les médias dans le menu de l'écran central.

  2. Choisissez TuneIn.

  3. Sélectionnez une station pour l'écouter.

  4. Appuyez sur Accueil pour afficher toutes les stations disponibles.

  5. Choisissez Récents, Favoris, Parcourirou Rechercher pour affiner les sélections.
