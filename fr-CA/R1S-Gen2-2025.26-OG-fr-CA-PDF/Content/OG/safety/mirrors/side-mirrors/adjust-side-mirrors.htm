Régler les rétroviseurs latéraux

  1. Accédez à votre profil de conducteur sur l’écran central.

  2. Choisissez les rétroviseurs latéraux.

  3. Utilisez les commandes au pouce et les boutons situés sur le côté gauche du volant pour régler le rétroviseur côté conducteur. Utilisez les commandes situées sur le côté droit du volant pour régler le rétroviseur côté passager.

  4. Choisissez « Mémoriser » pour enregistrer les paramètres. Une fois les paramètres enregistrés dans votre profil de conducteur, les rétroviseurs latéraux s'ajusteront automatiquement lorsque vous déverrouillerez le véhicule de l'extérieur.
