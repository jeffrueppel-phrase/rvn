Avec l’aimable autorisation du camp

Pour vous déplacer dans le véhicule sans déranger les autres, allumez Avec la courtoisie du camp Pour désactiver les éléments suivants :

  • Les lumières extérieures, comme les lumières qui s’allument automatiquement lorsque vous verrouillez ou déverrouillez le véhicule ou ouvrez les portes

  • Bruits du véhicule, tels que les bruits de serrure et les carillons extérieurs

  • Verrouillage ou déverrouillage par proximité

  • Gear Guard Alarme sonore

Lorsque Camp Courtesy est activé, utilisez la manette des phares pour allumer ou éteindre les phares.

Le véhicule règle les feux en mode automatique dans les conditions suivantes :

  • Vous désactivez Camp Courtoisie.

  • Vous conduisez plus vite que la vitesse de .

  • Le véhicule sort du mode veille et vous déplacez le véhicule hors de la P (Parc).

Lorsque le service de courtoisie du camp est actif, le véhicule peut limiter le chauffage ou la climatisation pour réduire le bruit ; Cependant, vous pouvez régler manuellement les commandes de climatisation pour accélérer le chauffage ou la climatisation.
