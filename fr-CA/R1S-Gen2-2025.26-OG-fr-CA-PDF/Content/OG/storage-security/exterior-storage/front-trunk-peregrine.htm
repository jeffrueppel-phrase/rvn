Utilisez le coffre avant

Assurez-vous qu'aucune personne ou objet ne se trouve sur le chemin du capot pendant qu'il s'ouvre ou se ferme, afin de réduire le risque de pincement ou d'autres blessures, ou d'endommager le véhicule.

Ouvrez ou fermez le capot pour accéder au coffre avant en utilisant l'un des éléments suivants :

  • Télécommande

  • Bouton sur la façade avant

  • Écran central

  • Bouton de coincement à l'intérieur du coffre avant

Ouvrir ou fermer le coffre avant avec la télécommande

Appuyez sur le bouton du coffre avant de la télécommande deux fois pour ouvrir ou fermer le capot.

Pour mettre en pause le capot pendant qu'il s'ouvre ou se ferme, appuyez une fois sur le bouton du coffre avant. Pour inverser la porte, appuyez à nouveau deux fois sur le bouton du coffre avant.

Ouvrir ou fermer le capot avec le bouton du coffre avant

Suivez ces étapes pour utiliser le bouton du coffre avant pour ouvrir le capot :

  1. Déverrouillez le véhicule.

  2. Appuyez une fois sur le bouton à l'avant du véhicule.

Avant de conduire le véhicule, fermez complètement le capot afin qu'il soit au même niveau que le véhicule. Si vous conduisez le véhicule avec le capot ouvert, le véhicule limite considérablement la vitesse maximale pour des raisons de sécurité.

Utilisez le bouton de piégeage du coffre avant

N'entrez pas dans le coffre avant. Il n'est pas destiné à contenir des personnes.

Le bouton de piégeage à l'intérieur du coffre avant brille dans l'obscurité pour une meilleure visibilité. Si quelqu'un est piégé à l'intérieur du capot, cette personne peut appuyer sur le bouton de piégeage pour ouvrir le coffre avant pendant que le véhicule est stationné. Le bouton de piégeage ne fonctionnera pas si les batteries sont déchargées. Reportez-vous aux instructions sur la façon de ouvrir le capot manuellement pour plus de détails.

Le bouton de piégeage ne libérera pas la porte du capot si le véhicule est en D (Conduite), N (Neutre) ou R (Marche arrière). Si quelqu'un appuie sur le bouton de piégeage, une notification apparaît sur l'affichage du conducteur. Préparez-vous à garer le véhicule immédiatement. Après avoir garé le véhicule, ouvrez le capot.
