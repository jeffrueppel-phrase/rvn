Afficher les vidéos enregistrées

Choisissez Gear Guard à partir du menu sur l'écran central, ou choisissez Vidéos de Gear Guard  pour afficher les vidéos enregistrées sous les types suivants :

  • Caméra de route : Déclenché lorsque vous choisissez Enregistrer le clip . La caméra de route enregistre également la vidéo et enregistre les clips de la caméra de route lorsque vous utilisez le klaxon pendant plus d'une seconde ou appuyez sur le bouton SOS . Le véhicule enregistre automatiquement les clips de la Road Cam sur votre périphérique de stockage externe.

    Pour visualiser les clips de la Road Cam dans l'onglet vidéo de la Road Cam, le périphérique de stockage externe doit être connecté à l'une des portes USB-C de la console centrale.

    Pour visualiser les enregistrements continus, accédez à votre périphérique de stockage externe depuis votre ordinateur et ouvrez le dossier Road Cam > Road Stream .

  • Incidents : Déclenchés par un certain événement de conduite tel qu'une collision ou un freinage brusque. Le véhicule enregistre automatiquement les vidéos d'incident sur le disque dur embarqué.
  • Cam de mouvement : Déclenché lorsque la Cam de mouvement détecte un mouvement lorsque vous êtes absent de votre véhicule ou déclenché par l'alarme si la Cam de mouvement est activée. Le véhicule enregistre automatiquement les vidéos de la Cam de mouvement sur le disque dur embarqué.

  • Épinglés : Liste les vidéos que vous avez sélectionnées pour qu'elles apparaissent sur cet onglet vidéo dans l'application Gear Guard. Les icônes suivantes distinguent les types de vidéos :

    • Icône de vidéo Road Cam clip

    • Icône de vidéo d'incident

    • Icône de vidéo Motion Cam

Le disque dur embarqué supprime automatiquement les vidéos Motion Cam après 10 jours. Pour enregistrer les vidéos de Motion Cam, choisissez Gear Guard à partir du menu sur l'écran central et choisissez Star  pour chaque vidéo que vous souhaitez sauvegarder. Si le disque dur embarqué est plein ou presque plein, le véhicule peut supprimer les anciennes vidéos de la caméra de mouvement, en commençant par les plus anciennes. Après la suppression des vidéos de la caméra de mouvement, si l'espace disponible ne suffit toujours pas pour de nouvelles enregistrements, le véhicule n'enregistrera pas de nouvelles vidéos d'incident ou de la caméra de mouvement.

Dans l'onglet Vidéo de la caméra de mouvement, les vidéos provenant du même emplacement sont regroupées dans un événement. Un marqueur bleu indique qu'un événement n'a pas encore été visionné.

Pour sélectionner plusieurs événements, choisissez Sélectionner , puis sélectionnez les événements.

Vous avez les options suivantes :

  • Choisissez Supprimer pour supprimer immédiatement les événements et leurs vidéos associées.

    Si vous supprimez une vidéo étoilée de la caméra de mouvement ou de l'alarme, vous la supprimez également de l'événement correspondant dans l'onglet Vidéo de la caméra de mouvement. Si vous supprimez une vidéo étoilée qui est un clip de Road Cam, vous la supprimez également de l'appareil de stockage externe.

  • Choisissez Annuler pour annuler vos modifications.

  • Choisissez Plus  pour ce qui suit :

    • Choisissez Sélectionner tout pour sélectionner tous les événements.

    • Choisir Copier sur le périphérique de stockage pour copier les événements sur votre périphérique de stockage externe.

      Vous ne pouvez pas copier les clips de Road Cam car ils sont automatiquement enregistrés sur votre appareil de stockage externe.

Lorsque vous consultez un événement dans l'application Gear Guard, la première vidéo qui se joue est celle dont la caméra a détecté le mouvement en premier.

Pour regarder une vidéo :

  1. Choisissez l'onglet vidéo pour Caméra de route, Incidents, Caméra de mouvement, ou Épinglé.

  2. Sélectionnez une vidéo ou un événement.

  3. Une image de véhicule affiche les différentes vues de caméra disponibles via les miniatures. Sélectionnez la vidéo que vous souhaitez regarder.

La vidéo se lit automatiquement dans le lecteur vidéo.

Vous pouvez naviguer entre les événements.

Lisez, mettez en pause, rembobinez ou avancez une vidéo. Vous avez les options suivantes :

  • Choisissez Télécharger  pour copier la vidéo sur votre périphérique de stockage externe.

  • Choisissez Star  pour afficher une vidéo dans l'onglet Vidéos épinglées ou pour enregistrer une vidéo de la caméra de mouvement et l'empêcher d'être automatiquement supprimée.

  • Choisissez Supprimer  pour supprimer une vidéo que vous avez sélectionnée dans un événement, ou pour supprimer toutes les vidéos d'un événement.

    La suppression d'une vidéo ou de toutes les vidéos d'un événement est permanente.
