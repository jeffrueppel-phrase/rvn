Acerca del Wi‑Fi del vehículo

El vehículo utiliza una conexión Wi‑Fi® para realizar diversas tareas, incluyendo las siguientes:

  • Descargar por aire (OTA) actualizaciones de software para el vehículo.

  • Conectarse de forma remota a la aplicación móvil de Rivian.

  • Acceder al mapa y buscar ubicaciones.

  • Transmitir desde las opciones de medios disponibles.

  • Habilitar diagnósticos remotos y descargar datos de registro.
