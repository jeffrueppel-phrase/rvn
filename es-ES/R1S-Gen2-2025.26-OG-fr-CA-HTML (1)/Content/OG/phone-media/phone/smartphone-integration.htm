Integración de Smartphone

Cuando emparejas tu teléfono con tu vehículo usando Bluetooth®, puedes elegir otorgar acceso a lo siguiente:

  • Llamadas telefónicas: Usa la pantalla central para realizar o recibir llamadas telefónicas.

  • Contactos: Usa la pantalla central para acceder a tu lista de contactos.

  • Audio multimedia: Transmite audio, como música o pódcast, desde tu teléfono al vehículo.

Elige Teléfono del menú en la pantalla central para acceder a lo siguiente:

  • Favoritos: Elige el icono Favoritos para acceder a una lista de tus contactos favoritos.

  • Llamadas recientes: Elige el icono Llamadas recientes para acceder a una lista de tus llamadas telefónicas recientes.

  • Contactos: Elige el icono Contactos para acceder a una lista de tus contactos.

  • Keypad: Elige el icono Teclado para mostrar un teclado y marcar números de teléfono.

  • Cambiar dispositivos: Elige el icono Cambiar dispositivos para mostrar una lista de dispositivos previamente conectados. Elige el dispositivo al que deseas cambiar, o elige Bluetooth para acceder a la configuración de Bluetooth.

Favoritos

La pantalla central muestra tus contactos favoritos.

Elige un contacto favorito para hacer una llamada inmediata y mostrar la información del contacto.

Usa tu teléfono, no la pantalla central, para designar un contacto como favorito.

Llamadas recientes

La pantalla central muestra lo siguiente:

  • Llamadas del historial de llamadas del teléfono conectado

  • Llamadas que has realizado usando el teclado desde la pantalla central

El icono Llamada saliente aparece frente a cualquier llamada que hayas realizado.

Accede a cada llamada reciente de la siguiente manera:

  • Elige el contacto o número de teléfono, si está disponible, para hacer una llamada inmediata.

  • Elige el Información icono para mostrar la información de contacto, si está disponible.

Contactos

Para gestionar tus contactos:

  1. Busca un contacto o selecciona una letra de la columna del alfabeto.

    • Elige Buscar contactos para mostrar un teclado.

    • Elige el Ordenar icono para ordenar los contactos por nombre o apellido.

  2. Selecciona un contacto.

  3. La pantalla central muestra la información de contacto disponible, que incluye números de teléfono y direcciones.

    Elige un número de teléfono para hacer una llamada inmediata, o elige una dirección para abrirla en la aplicación de Navegación.

  • Usa tu teléfono, no la pantalla central, para añadir un contacto.

  • Si Contactos está desactivado, accede a tu teléfono y permite que comparta tus contactos.

Media Audio

Transmite audio, como música o pódcast, desde tu teléfono al vehículo. Elige la fuente de medios en tu teléfono.
