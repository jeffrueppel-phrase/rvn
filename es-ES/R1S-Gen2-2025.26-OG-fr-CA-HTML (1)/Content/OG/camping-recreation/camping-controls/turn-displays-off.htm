Desactivar pantallas

Elegir Desactivar pantallas para apagar las pantallas central, del conductor y traseras.

Para encender las pantallas, toca la pantalla central. Si no se encienden, pise el pedal del freno. Si ninguna de las acciones enciende las pantallas, el vehículo está dormido. Desbloquee el vehículo y presione el pedal del freno para activarlo y encender las pantallas.
