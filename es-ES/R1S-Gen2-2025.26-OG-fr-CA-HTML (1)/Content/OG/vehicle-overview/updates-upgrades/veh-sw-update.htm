Actualización del software del vehículo

Puede mejorar la funcionalidad de su vehículo actualizando su software a través de una conexión Wi-Fi® o conexión celular.

Cuando haya una actualización de software disponible para su vehículo, el vehículo le notificará y proporcionará información sobre el estado, como la siguiente:

  • Número de versión de la actualización de software

  • Tiempo estimado que tomará la actualización de software

Elija Ver novedades para leer los detalles de la actualización.

La notificación de actualización de software ofrece tres opciones de instalación:

  • Instalar ahora para iniciar la actualización del software.

  • Programar para más tarde para elegir una hora en el futuro para esta actualización o para todas las actualizaciones futuras.

  • No ahora para borrar el mensaje y no instalar la actualización.

Cuando eliges Programar para más tarde, tú:

  • Seleccionas una hora de instalación en el selector.

  • Opcionalmente, puede activar Instalar automáticamente siempre a [hora] para configurar la instalación automática de futuras actualizaciones de software. Puede activar o desactivar esta configuración en Configuración  > Vehículo > Actualizaciones en la pantalla central.

  • Tenga la opción de configurar Solo instalar en casa para la instalación automática.

Si has configurado previamente la instalación automática, la notificación muestra la hora programada y te permite elegir:

  • Instalar ahora para ejecutar la instalación de inmediato.

  • Cambiar el tiempo de instalación para seleccionar un tiempo diferente.

  • Entendido Para aceptar el horario programado anteriormente.

  • Con la instalación automática, el tiempo real de instalación puede variar unos minutos.

  • El sistema permite al menos 8 horas entre la descarga del software y la instalación para asegurarse de que está programado.

Para acceder o cambiar la configuración del software o ver el progreso de una actualización de software, vaya a Ajustes > Vehículo > Actualizaciones en la pantalla central.

Antes de actualizar el software de su vehículo, asegúrese de que:

  • La batería tenga al menos 35 millas de autonomía.

  • El vehículo esté en modo Estacionamiento.

  • El vehículo no esté conectado a un cargador rápido de corriente continua.

  • El modo de cambio de neumáticos esté apagado.

  • Las temperaturas del sistema no sean demasiado altas o demasiado bajas.

  • La batería de 12 V esté cargada y en buen estado.

  • La comodidad para mascotas está apagada.

  • El nivel del vehículo está apagado.

  • Todos los pasajeros han salido del vehículo (sugerido).

Si hay una nueva actualización del sistema disponible pero no se cumplen alguno de los requisitos anteriores, recibirá una notificación en la que podrá elegir:

  • Vaya a Actualizaciones para obtener más información.

  • Entendido para cerrar la notificación.

Mientras su vehículo actualiza el software, no toque los pedales ni los botones. De lo contrario, esto puede interrumpir la actualización e inutilizar el vehículo.

El acceso al almacenamiento, el bloqueo y las funciones de seguridad no están disponibles mientras el vehículo actualiza el software.

Si el vehículo no completa la actualización del software, consulte la pantalla central para obtener información adicional.

Actualizaciones que no son de firmware

Las actualizaciones que no son de firmware no requieren ningún tiempo de inactividad ni un reinicio del sistema. Siempre que tenga una conexión Wi-Fi, estas actualizaciones se pueden recibir y descargar fácilmente.

  1. Si no está conectado a Wi-Fi cuando una actualización que no es de firmware esté disponible, recibirá una notificación en la pantalla central.

  2. Selecciona Conectar a Wi-Fi para recibir la descarga o Hazlo más tarde para posponerla.

  3. Si está conectado a Wi-Fi, la actualización aparece en Vehículo > Actualizaciones en la pantalla central.

  4. Seleccionar Conectar a Wi-Fi.

  5. Elija una red Wi-Fi e introduzca su contraseña, si se requiere.

  6. El sistema descarga automáticamente la actualización durante el siguiente ciclo de reposo del vehículo.
