Gestionar sonidos de bloqueo

Cuando el vehículo detecta un comando de bloqueo de una llave fuera del vehículo, confirma la acción de bloqueo con un sonido. Los sonidos disponibles incluyen el trino de un mirlo azul (el predeterminado), el ulular de un búho, o una combinación de ambos. El sonido de bloqueo está activado por defecto.

Para cambiar el sonido de bloqueo o para activarlo o desactivarlo:

  1. Ve a Configuración > Vehículo > Acceso en la pantalla central.

  2. Bajo Sonido de bloqueo (exterior), selecciona uno de los siguientes:

    1. Mirlo azul para usar el trino del mirlo azul.

    2. búho para usar el ulular del búho.

    3. Día/Noche para usar el trino del mirlo azul durante las condiciones de luz diurna, y el ulular del búho durante las condiciones de luz nocturna.

    4. El botón de alternancia para activar o desactivar el sonido de bloqueo.
