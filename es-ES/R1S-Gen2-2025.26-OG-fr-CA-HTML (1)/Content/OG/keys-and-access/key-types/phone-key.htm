Tecla del teléfono

Para usar tu teléfono para bloquear o desbloquear el vehículo, utiliza uno de los siguientes métodos:

  • Selecciona el Bloquear o Desbloquear iconos en la aplicación móvil de Rivian.

  • Usa Bloqueo por proximidad cuando estés cerca del vehículo.

Puedes emparejar hasta cuatro Bluetooth® smartphones de baja energía con el vehículo. Consulta Aplicación móvil de Rivian para obtener instrucciones.
