Copia un video

Para copiar un video del disco duro interno a tu computadora, utiliza tu dispositivo de almacenamiento externo (no incluido).

Este puede ser el mismo dispositivo de almacenamiento externo utilizado para la cámara de carretera.

El dispositivo de almacenamiento externo requiere lo siguiente:

  • 128 GB (mínimo) de almacenamiento

    El tamaño máximo de almacenamiento es de 2 TB.

  • Compatibilidad con USB 3.x y USB 2.0

  • Se recomienda unidad de estado sólido (SSD)

  • 20 MB/s (mínimo) de velocidad de escritura aleatoria

  • Sistema de archivos FAT32

    El vehículo solicita reformatear tu dispositivo si detecta un sistema de archivos diferente.

Para copiar un video a tu dispositivo de almacenamiento externo, sigue estas instrucciones:

  1. Conecta el dispositivo de almacenamiento externo a uno de los puertos USB-C en la consola central de tu vehículo.

    No uses un hub o divisor USB.

  2. Puede aparecer un aviso en la pantalla central. Elige Formatear dispositivo para borrar todos los datos y formatear el dispositivo de almacenamiento externo para su uso por la aplicación Gear Guard.

    El vehículo puede tardar unos minutos en formatear el dispositivo de almacenamiento externo.

  3. Elige Gear Guard del menú en la pantalla central.

  4. Selecciona el evento y luego selecciona el video para descargar.

  5. Elige Descargar  para copiar el video a tu dispositivo de almacenamiento externo.

  6. El vehículo guarda el video en tu dispositivo de almacenamiento externo.

Para quitar de forma segura el dispositivo de almacenamiento externo del vehículo, elige Gear Guard del menú en la pantalla central. Elige Características de Gear Guard , y luego elige Expulsar.
