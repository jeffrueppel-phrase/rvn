Control de Crucero Manual

El Control de Crucero Manual solo está disponible cuando estás remolcando un remolque. El Control de Crucero Manual mantiene una velocidad establecida y te permite quitar el pie del pedal del acelerador.

  • Cuando el Control de Crucero Manual está activado, siempre monitorea el tráfico y las condiciones de conducción, y prepárate para frenar si es necesario. Esta función, junto con la configuración de activación de velocidad, está diseñada para apoyarte, el conductor, y no reemplaza tu atención, juicio y necesidad de controlar el vehículo.

  • El Control de Crucero Manual no ajusta la velocidad establecida de acuerdo con los límites de velocidad publicados. Es tu responsabilidad ajustar la velocidad establecida de acuerdo con el límite de velocidad publicado.

Para activar el Control de Crucero Manual, tira de la palanca de conducción hacia ti y luego suéltala. El vehículo emite un sonido y luego mantiene la velocidad establecida. La velocidad establecida aparece en un círculo en la parte superior de la pantalla del conductor.

Para desactivar el Control de Crucero Manual, empuja la palanca de conducción lejos de ti y luego suéltala, o presiona el pedal del freno.

El Control de Crucero Manual se desactiva cuando aplicas el Freno Manual del Remolque o si ocurre un evento de oscilación del remolque. Presiona el pedal del freno o el Freno Manual del Remolque para reducir la velocidad del vehículo.

Cuando el Control de Crucero Manual está activado, siempre monitorea el tráfico y las condiciones de conducción. Prepárate para presionar el pedal del freno o del acelerador si es necesario.

  • El Control de Crucero Manual no mantendrá una distancia apropiada de un vehículo que esté delante de ti en el carril actual.

  • Si conduces cuesta arriba o cuesta abajo mientras usas el Control de Crucero Manual, el vehículo puede no mantener la velocidad establecida.
