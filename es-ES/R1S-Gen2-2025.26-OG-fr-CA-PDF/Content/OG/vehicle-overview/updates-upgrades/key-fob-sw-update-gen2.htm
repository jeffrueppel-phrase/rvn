Actualización del software del mando a distancia

Mejore la funcionalidad del mando a distancia actualizando su software a través de Wi-Fi® o conexión celular.

Cuando haya una actualización de software disponible para el mando a distancia, el vehículo le notificará y proporcionará información sobre el estado, como el tiempo estimado que tomará la actualización del software.

Para acceder a la configuración del software, vaya a Ajustes > Vehículo > Actualizaciones en la pantalla central.

Antes de actualizar el software del mando a distancia, haga lo siguiente:

  • Estacione el vehículo.

  • Asegúrese de tener una llave adicional como copia de seguridad. El mando a distancia no funcionará mientras se actualice el software.

Elija Actualizar ahora para iniciar la actualización.

Mientras el mando a distancia actualiza el software, manténgalo en el vehículo.

Si el mando a distancia no completa la actualización del software, consulte la pantalla central para obtener información adicional.
