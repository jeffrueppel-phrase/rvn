Actualización de software del vehículo

Puedes mejorar la funcionalidad de tu vehículo actualizando su software a través de una conexión Wi-Fi® o de datos móviles.

Cuando hay una actualización de software disponible para tu vehículo, este te notifica y proporciona información de estado, como la siguiente:

  • Número de versión de la actualización de software

  • Tiempo estimado que tomará la actualización de software

Elige Ver novedades para leer los detalles de la actualización.

La notificación de actualización de software ofrece tres opciones de instalación:

  • Instalar ahora para comenzar la actualización de software.

  • Programar para más tarde para elegir un momento en el futuro para esta actualización o para todas las actualizaciones futuras.

  • No ahora para eliminar el mensaje y no instalar la actualización.

Cuando eliges Programar para más tarde, tú:

  • Selecciona un momento de instalación en el selector.

  • Opcionalmente, puedes activar Siempre instalar automáticamente a [hora] para configurar la instalación automática de futuras actualizaciones de software. Puedes activar o desactivar esta configuración en Configuración  > Vehículo > Actualizaciones en la pantalla central.

  • Tienes la opción de configurar Solo instalar en casa para la instalación automática.

Si has configurado previamente la instalación automática, la notificación muestra la hora programada y te permite elegir:

  • Instalar ahora para ejecutar la instalación de inmediato.

  • Cambiar hora de instalación para seleccionar un momento diferente.

  • Entendido para aceptar la hora programada previamente.

  • Con la instalación automática, el tiempo de instalación real puede variar unos minutos.

  • El sistema permite al menos 8 horas entre la descarga del software y la instalación para asegurarse de que esté al tanto de que está programado.

Para acceder o cambiar la configuración del software o ver el progreso de una actualización de software, vaya a
Configuración > Vehículo > Actualizaciones en la pantalla central.

Antes de actualizar el software de su vehículo, asegúrese de que:

  • La batería tiene al menos 35 millas de autonomía.

  • El vehículo está en Parking.

  • El vehículo no está conectado a un cargador rápido de CC.

  • El modo de cambio de neumáticos está desactivado.

  • Las temperaturas del sistema no son demasiado altas ni demasiado bajas.

  • La batería de 12 V está cargada y en buen estado.

  • La comodidad de las mascotas está desactivada.

  • El nivelado del vehículo está desactivado.

  • Todos los pasajeros han salido del vehículo (sugerido).

Si hay una nueva actualización del sistema disponible pero no se cumplen alguno de los requisitos anteriores, recibirá una notificación donde podrá elegir:

  • Ir a Actualizaciones para aprender más.

  • Entendido para cerrar la notificación.

Mientras su vehículo actualiza el software, no toque los pedales ni los botones. De lo contrario, esto puede interrumpir la actualización y deshabilitar el vehículo.

El acceso al almacenamiento, el bloqueo y las funciones de seguridad no están disponibles mientras el vehículo actualiza el software.

Si el vehículo no completa la actualización del software, verifica la pantalla central para obtener información adicional.

Actualizaciones no de firmware

Las actualizaciones no de firmware no requieren tiempo de inactividad ni reinicio del sistema. Siempre que tengas una conexión Wi-Fi, estas actualizaciones se pueden recibir y descargar fácilmente.

  1. Si no estás conectado a Wi-Fi cuando una actualización no de firmware esté disponible, recibirás una notificación en la pantalla central.

  2. Selecciona Conectar a Wi-Fi para recibir la descarga o Hacer esto más tarde para posponerla.

  3. Si estás conectado a Wi-Fi, la actualización aparece en Vehículo > Actualizaciones en la pantalla central.

  4. Selecciona Conectar a Wi-Fi.

  5. Elige una red Wi-Fi e ingresa tu contraseña, si es necesario.

  6. El sistema descarga automáticamente la actualización durante el próximo ciclo de sueño del vehículo.
