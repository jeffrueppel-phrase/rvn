Apagar pantallas

Seleccione Apagar pantallas para apagar las pantallas central, del conductor y trasera.

Para encender las pantallas, toque la pantalla central. Si no se encienden, presione el pedal del freno. Si ninguna de estas acciones enciende las pantallas, el vehículo está inactivo. Desbloquee el vehículo y presione el pedal del freno para activarlo y encender las pantallas.
