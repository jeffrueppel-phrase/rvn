Use el punto de acceso Wi-Fi

Controla el Wi-Fi® punto de acceso y acceda a la configuración a través de lo siguiente:

  • Ícono del punto de acceso Wi-Fi en la barra de estado en la pantalla central

  • Aplicación de configuración en la pantalla central

El ícono del punto de acceso Wi-Fi indica el estado del punto de acceso Wi-Fi.

Icono

Descripción

El punto de acceso Wi-Fi está activado.

El punto de acceso Wi-Fi está apagado.

  1. Elija el icono Punto de acceso Wi-Fi . , o vaya a Configuración > Conectar > Punto de acceso.

  2. Girar Punto de acceso activado o desactivado.

Si el punto de acceso está activado, consulte la contraseña, el nombre del punto de acceso y el número de dispositivos conectados en la pantalla central. En el dispositivo que desea conectar al punto de acceso Wi-Fi, siga estas instrucciones:

  1. Vaya a la configuración de Wi-Fi.

  2. Seleccione el nombre del punto de acceso.

  3. Introduzca la contraseña que aparece en la pantalla central.

  4. Elegir Únete.

  • Si sus dispositivos pierden la conexión al punto de acceso Wi-Fi, consulte la pantalla central para obtener más información.

  • El vehículo apaga automáticamente el punto de acceso Wi-Fi cuando sales del vehículo.

  • El punto de acceso Wi-Fi solo está disponible cuando el vehículo tiene conectividad LTE. Si el vehículo pierde la conectividad LTE, apaga automáticamente el punto de acceso Wi-Fi.
