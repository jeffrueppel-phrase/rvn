Acceder a la configuración de Wi‑Fi

Accede a la configuración de Wi‑Fi® a través de una de las siguientes opciones:

  • Elige el icono de Wi‑Fi y luego elige Configuración .

  • Ve a Configuración > Conectar > Wi‑Fi.

Si tienes una red Wi‑Fi guardada, puedes seleccionar esa red para unirte automáticamente. También puedes encontrar una red o eliminar cualquier red que ya no quieras guardar.

El vehículo no muestra un icono de bloqueo junto a una red no segura; esto indica que la red puede ser un riesgo de seguridad. Asegúrate de conectar el vehículo a una red segura.

Encontrar una red Wi‑Fi

  1. Ve a Configuración > Conectar > Wi‑Fi.

  2. Elige Buscar red.

  3. Introduce el nombre de la red.

    • El nombre de la red es sensible a mayúsculas y minúsculas.

    • El vehículo no admite la conexión a SSIDs de Wi-Fi que contengan caracteres de escape, nueva línea, retorno de carro o tabulación.

  4. Elige Siguiente.

  5. Introduce la contraseña.

    • La contraseña es sensible a mayúsculas y minúsculas.

    • Elige el botón Mostrar para ver la contraseña.

  6. Elige Unirse.

  • Asegúrate de que tu router o extensor utilice el acceso protegido por Wi‑Fi 2-Estándar de cifrado avanzado (WPA2‑AES) para la seguridad inalámbrica. El vehículo no admite cifrado WPA3. El vehículo no admite el Protocolo de Integridad de Clave Temporal (TKIP) o la Privacidad Equivalente por Cable (WEP), por lo que no mostrará ni se conectará a redes que utilicen TKIP o WEP.

  • Asegúrate de que la red Wi‑Fi no utilice un portal cautivo, que es utilizado por algunos cafés u hoteles para gestionar el acceso a Wi‑Fi. El vehículo puede conectarse inicialmente, pero no admitirá el acceso a internet a través de un portal cautivo. Si has guardado previamente una red de portal cautivo, elimínala de la lista de redes guardadas para evitar una conexión automática.

Eliminar una red Wi‑Fi guardada

  1. Ve a Configuración > Conectar > Wi‑Fi.

  2. Elige Gestionar redes.

  3. Elige el botón Eliminar para eliminar inmediatamente una red Wi‑Fi guardada.

  4. Elige Eliminar.

  5. Elige Hecho.
