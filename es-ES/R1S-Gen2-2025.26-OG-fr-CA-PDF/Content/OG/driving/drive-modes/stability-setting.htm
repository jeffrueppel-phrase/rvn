Estabilidad

Elige una configuración de Estabilidad basada en el modo de conducción.

  • Activado: El Control Electrónico de Estabilidad (ESC) y el Sistema de Control de Tracción (TCS) proporcionan la máxima estabilidad y tracción en todas las condiciones de conducción.

  • Reducido: Tanto el ESC como el TCS se activan para ayudar a estabilizar el vehículo. Reducido permite una pequeña cantidad de deslizamiento de las ruedas.

  • Apagar: Desactiva completamente el ESC y mantiene el TCS en su configuración menos restrictiva. Apagar permite el mayor nivel de deslizamiento de las ruedas.
