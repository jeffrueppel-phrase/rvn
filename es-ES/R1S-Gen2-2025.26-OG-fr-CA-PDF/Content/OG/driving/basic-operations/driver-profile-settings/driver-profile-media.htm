Aplicación multimedia favorita

El vehículo guarda las últimas aplicaciones multimedia utilizadas en el perfil del conductor. Cuando reconoce una ID de Rivian al inicio de un viaje, la aplicación multimedia guardada del viaje anterior se carga automáticamente.

Para los suscriptores de Connect+, las aplicaciones guardadas incluyen Spotify®, Apple Music, Audible, TIDAL®y TuneIn®.
