Control de velocidad manual

El control de velocidad manual solo está disponible cuando remolque un remolque. El control de velocidad manual mantiene una velocidad establecida y le permite quitar el pie del pedal del acelerador.

  • Cuando se activa el control de velocidad manual, siempre debe controlar el tráfico y las condiciones de conducción, y estar preparado para frenar si es necesario. Esta función, junto con la configuración de activación de la velocidad, está diseñada para ayudarle, al conductor, y no sustituye su atención, juicio y necesidad de controlar el vehículo.

  • El control de velocidad manual no ajusta la velocidad establecida según los límites de velocidad establecidos. Es su responsabilidad ajustar la velocidad establecida de acuerdo con el límite de velocidad publicado.

Para activar el control de velocidad manual, tire del mando de conducción hacia usted y luego suelte. El vehículo emite un pitido y luego mantiene la velocidad establecida. La velocidad establecida aparece en un círculo en la parte superior de la pantalla del conductor.

Para desactivar el control de velocidad manual, empuje el mando de conducción lejos de usted y luego suelte, o presione el pedal del freno.

El control de velocidad manual se desactiva cuando aplica el freno manual del remolque o si ocurre un evento de balanceo del remolque. Presione el pedal de freno o el freno manual del remolque para reducir la velocidad del vehículo.

Cuando se active el control de crucero manual, siempre debe controlar el tráfico y las condiciones de conducción. Prepárese para presionar el pedal de freno o acelerador si es necesario.

  • El control de crucero manual no mantendrá una distancia adecuada de un vehículo que va delante de usted en el carril actual.

  • Si conduce cuesta arriba o cuesta abajo mientras utiliza el control de crucero manual, es posible que el vehículo no mantenga la velocidad establecida.
