Usa el maletero delantero

Asegúrate de que no haya personas u objetos en el camino del capó mientras se abre o se cierra, para reducir el riesgo de pellizcos u otras lesiones, o daños al vehículo.

Abre o cierra el capó para acceder al maletero delantero usando uno de los siguientes:

  • Llave inteligente

  • Botón en la fascia delantera

  • Pantalla central

  • Botón de atrapamiento dentro del maletero delantero

Abre o cierra el maletero delantero con la llave inteligente

Presiona el botón del maletero delantero en la llave inteligente dos veces para abrir o cerrar el capó.

Para pausar el capó mientras se abre o se cierra, presiona el botón del maletero delantero una vez. Para revertir la puerta, presiona el botón del maletero delantero dos veces más.

Abre o cierra el capó con el botón del maletero delantero

Sigue estos pasos para usar el botón del maletero delantero para abrir el capó:

  1. Desbloquea el vehículo.

  2. Presiona el botón en la fascia delantera una vez.

Antes de conducir el vehículo, cierra el capó completamente para que esté al ras con el vehículo. Si conduces el vehículo con el capó abierto, el vehículo limita significativamente la velocidad máxima por razones de seguridad.

Usa el botón de atrapamiento del maletero delantero

No entres en el maletero delantero. No está destinado a sostener personas.

El botón de atrapamiento dentro del maletero delantero brilla en la oscuridad para mayor visibilidad. Si alguien está atrapado dentro del capó, esa persona puede presionar el botón de atrapamiento para abrir el maletero delantero mientras el vehículo está estacionado. El botón de atrapamiento no funcionará si las baterías están muertas. Consulta las instrucciones sobre cómo abrir el capó manualmente para más detalles.

El botón de atrapamiento no liberará la puerta del capó si el vehículo está en D (Conducir), N (Neutro) o R (Reversa). Si alguien presiona el botón de atrapamiento, aparece una notificación en la pantalla del conductor. Prepárate para estacionar el vehículo de inmediato. Después de estacionar el vehículo, abre el capó.
