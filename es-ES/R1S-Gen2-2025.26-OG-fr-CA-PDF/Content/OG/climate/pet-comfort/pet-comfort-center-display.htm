Pantalla central

Para activar la comodidad para mascotas:

  1. Elige Ventilador desde el menú en la pantalla central.

  2. En el panel de clima, elige Comodidad para mascotas .

  3. Elige Activar, o elige Cancelar si decides no usar la comodidad para mascotas.

Para desactivar la comodidad para mascotas, conduce el vehículo o elige Comodidad para mascotas en el panel de clima.

Cuando la pantalla central indica que la comodidad para mascotas está activa (tu mascota está segura y cómoda), los botones de las puertas abiertas y los interruptores de las ventanas están deshabilitados.

  • Si abres las ventanas y luego activas la comodidad para mascotas, las ventanas permanecen abiertas.

  • La pantalla central o trasera proporciona instrucciones para habilitar temporalmente los botones de las puertas abiertas si alguien está encerrado dentro. O presiona rápidamente un botón de puerta abierta cinco veces para habilitar la puerta.

  • Puedes activar la comodidad para mascotas cuando el rango de la batería es mayor que .

  • Después de activar la comodidad para mascotas, permanece activa hasta que la batería se agote.

  • Debes desactivar la comodidad para mascotas antes de comenzar cualquier actualización de software.
