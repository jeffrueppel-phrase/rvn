Retractor de bloqueo automático (ALR)

El retractor de bloqueo automático (ALR) hace que el cinturón de seguridad se bloquee cuando se tira muy rápidamente. El ALR se puede usar para asegurar un asiento para niños.

Activar ALR

  1. Saque completamente el cinturón de seguridad.

  2. Retraiga el cinturón de seguridad a la longitud deseada para activar el bloqueo del cinturón de seguridad en esa posición. Una vez bloqueado, el cinturón de seguridad no se puede extender más.

Cancelar ALR

Restaure el cinturón de seguridad a la función normal desabrochando y retrayendo completamente el cinturón de seguridad. Cancelar ALR al quitar un asiento para niños.

Los ALR son diferentes de los retractores de bloqueo de emergencia (ELR), que están diseñados para permitir que los ocupantes se muevan libremente una vez que el cinturón de seguridad está abrochado, pero se bloquean en caso de accidente. Los ALR solo están equipados en los asientos de pasajeros.
