Teclado del teléfono

Para usar el teléfono para bloquear o desbloquear el vehículo, utilice uno de los siguientes métodos:

  • Seleccione el Bloqueo o Desbloquear en la aplicación móvil Rivian.

  • Uso Bloqueo de proximidad cuando esté cerca del vehículo.

Puede emparejar hasta cuatro Bluetooth® Smartphones de baja energía con el vehículo. Consulte la Aplicación móvil de Rivian para obtener instrucciones.
