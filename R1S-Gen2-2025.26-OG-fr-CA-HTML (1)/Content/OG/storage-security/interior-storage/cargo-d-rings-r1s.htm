Cargo D-Rings

In some Model Year 2026 vehicles, cargo D-rings are available instead of cargo tie-downs.

Use the cargo D-rings, if equipped, to prevent cargo from moving and sliding. These cargo D-rings are located in each corner of the cargo area floor. They can be used to tie down items with a rope.

These D-rings have been designed with a weight rating factor of per tie-down.

Lift the hook on each cargo D-ring to adjust the angle of the D-ring.
