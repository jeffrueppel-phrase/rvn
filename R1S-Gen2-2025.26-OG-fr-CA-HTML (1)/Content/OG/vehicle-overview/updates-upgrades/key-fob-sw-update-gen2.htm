Key Fob Software Update

Improve the functionality of the key fob by updating its software over a Wi-Fi® or cellular connection.

When a software update is available for the key fob, the vehicle notifies you and provides status information, such as the estimated time the software update will take.

To access software settings, go to Settings > Vehicle > Updates on the center display.

Before updating the software for the key fob, do the following:

  • Park the vehicle.

  • Ensure that you have an extra key with you as a backup. The key fob won't work while the software updates.

Choose Update Now to start the update.

While the key fob updates the software, keep it in the vehicle.

If the key fob doesn't complete the software update, check the center display for additional information.
