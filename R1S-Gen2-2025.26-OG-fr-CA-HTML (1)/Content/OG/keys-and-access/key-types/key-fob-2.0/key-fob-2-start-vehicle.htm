Start the Vehicle with a Key Fob

To drive the vehicle without a phone key, start it within 60 seconds of unlocking the vehicle with the key fob. If 60 seconds pass before you start the vehicle, touch the key fob to the charging tray on top of the center console for a few seconds. The vehicle chimes to confirm. Once you hear the chime, you can start the vehicle.

If the key fob battery is dead, lay the key fob on the charging tray (button side up) until you hear the chime.
