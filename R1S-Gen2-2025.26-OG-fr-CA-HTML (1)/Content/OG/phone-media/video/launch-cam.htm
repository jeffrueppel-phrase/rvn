Launch Cam

If you want to record launch events in Launch Mode, you can set it and view the videos in the Video app.

  1. To turn Launch Cam on or off, go to Video > Launch Cam.

  2. Select Record video during launch events to turn on Launch Cam.

  3. Select Set up now.

    4. You must have an external storage device to use Launch Cam. This isn't necessary if you have already added a storage device for Road Cam.

  4. You can select Go to drive modes to use Launch Mode or other drive modes.
