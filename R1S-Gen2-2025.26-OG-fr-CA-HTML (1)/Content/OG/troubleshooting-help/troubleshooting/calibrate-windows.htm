Windows

Keep the path of the window clear and free of obstructions while resetting the windows.

If a window isn't going all the way up or down, the Rivian mobile app says that it's open while it's closed, or it's bouncing back down when it reaches the top, reset the window position.

  1. Move the window all the way to the top (or bottom) position using the first notch of the window switch.

  2. Release the window switch.

  3. Hold the window switch up (or down) to the first notch for at least 8 seconds. This starts the window position reset process.

  4. Repeat steps 1‑3 for any window that isn't working normally.

If a window doesn't express close or you get a notification that says it needs calibration, reset the window calibration.

  1. Lower the window until it completely opens.

    If needed, unlock the passenger windows. Go to Vehicle  in the status bar on the center display. Then choose Shortcuts > Window Locks.

  2. Hold the window switch down for an additional 5 seconds.

  3. Raise the window until it completely closes. If the window stops on the way up, return to step 2.

  4. Hold the window switch up for an additional 5 seconds.

  5. Ensure that express close works from the window switch.

  6. Repeat steps 1‑5 for each window that needs calibration.
